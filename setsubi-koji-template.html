リスク評価方法

R = S（重篤度 1-4） × F（頻度 1-4） × D（回避難度 1-4）
低（1-12） 中（13-24） 高（25-40） 極高（41-64）
重篤度S: 1=軽傷／2=休業／3=後遺症／4=死亡。頻度F: 1=年1回／2=月数回／3=週数回／4=毎日。回避難度D: 1=容易／2=可能／3=困難／4=不可能。

全エネルギー源リスト（必須・5分で書き切る）

「停止＝ゼロエネルギー」の思考停止を防ぐ。隠れたエネルギー源（UPS・残液・残熱・誘起電圧）の見落としを構造的に防止する欄。

MOC・前回工事差分・運転側追記

装置更新・DCS更新・配管経路変更は MOC 対象。未取得で工事完了→運転再開は重大コンプラ違反。
「過去にやったから大丈夫」の思考停止防止。自社プラント連続プロセスは前回と残留物・運転条件が異なるのが普通。
協力会社RA丸投げ防止。現場暗黙知（弁のクセ・運転員特性・図面齟齬）は運転側しか知らない。
図面齟齬を「現場が間違い」と決めつける罠の防止。事前合意でトラブル時の判断遅延を防ぐ。

1. リスクアセスメントの基本思想（ISO 12100 / JIS B 9700）

3ステップメソッド（優先順）:

  1. 本質的安全設計: ハザードそのものを除去・低減（無圧化・冷却・絶縁設計）
  2. 安全防護および付加保護方策: ガード・インタロック・遮断器・接地
  3. 使用上の情報: 警告ラベル・KY・教育・PPE
PPE（保護具）はあくまで「最終手段」。PPE頼みのRAは不十分。設計段階での①②検討の証跡を残すこと。

2. 残存エネルギー5原則（撤去工事の必読項目）

5種類のエネルギー

  • 圧力: 配管・タンク・ボンベ・蓄圧器
  • 電気: 電源／コンデンサ蓄電／誘導電圧／UPS・非常電源
  • 熱: 高温配管／低温（液窒・冷媒）
  • 化学: 残液・残ガス・反応性物質・触媒
  • 位置／運動: 吊り荷・バネ・ばね蓄勢

縁切りの優先順位

  1. 物理切断（配管：盲板挿入／ケーブル：切断＋絶縁テープ）
  2. 二重弁＋ベント（DBB: Double Block & Bleed）
  3. 単弁＋札掛け（LOTO: Lockout/Tagout）
  4. 札掛けのみ（最低限／推奨せず）

3. 関連法令体系

労働安全衛生分野

  • 労働安全衛生法 第28条の2（RA努力義務）／第57条（表示）
  • 安衛則 第36条（特別教育）／第151条（クレーン）／第518条（高所）
  • 特化則: 特定化学物質取扱（特管／第3類）
  • 有機則: 有機溶剤55物質
  • 石綿則: アスベスト含有材の解体・除去
  • 酸欠則: 酸欠危険場所（第一種／第二種）
  • 電離則: 放射線取扱（該当時）

設備・物質関連

  • 電気事業法 第43条（主任技術者）／第48条（工事計画届）
  • 電技解釈: 接地・離隔・絶縁の具体基準
  • 高圧ガス保安法: 容器・配管・冷凍／第14条許可
  • 消防法 危険物（第4類等）・指定数量
  • 廃棄物処理法 産廃／特管（PCB・石綿）／マニフェスト
  • PCB特措法: PCB廃棄物の処分期限
  • フロン排出抑制法: SF6・冷媒の回収

4. KY（危険予知）実施タイミング

  • 必須 TBM-KY: 当日朝、作業開始前（Tool Box Meeting）
  • 必須 工程変更時: 想定外発生・手順変更・人員交代
  • 必須 作業中断後の再開時: 昼休憩後・翌日朝
  • 推奨 協力会社入替時: 引継ぎ部分の認識合わせ
  • 推奨 ヒヤリハット直後: 当事者だけでなく作業班全体で

5. 用語混同チェック（届出 / 許可 / 認可 / 報告 / 通知）

用語性質例
届出事前提出のみ・行政の応答不要労基署 機械等設置届／電気主任技術者選任届
許可行政の判断あり・許可なしは違法高圧ガス製造許可／産廃収集運搬業許可
認可第三者行為の効力を補完料金認可（電気・ガス）
報告事後・定期的労災事故報告／PCB処分状況報告
通知関係者への一方的伝達停電作業通知／工事工程通知
日常会話で混用される5語。RA／工事計画書では正確に使い分けること。「届出 = 役所に出すだけでOK」と「許可 = 判断待ち」の差は工程遅延に直結する。

6. 運用ルール

  1. 本テンプレを基に、案件着手時に複製してプロジェクト名を入れる
  2. 案件タイプフィルタで該当タイプを選択（複数選択可）。共通項目は常時表示
  3. RAは 計画段階・着工前・完工時 の3回見直す
  4. 残留リスクは「低」になるまで対策を追加（「中」止まりは原則NG）
  5. チェックリストは「再開発困難な状態」（縁切り・残圧抜き）を 個別に確認者サインを残す 運用が望ましい
  6. 完了後は本HTMLをPDF化して工事記録に保存

7. 案件タイプ別の留意点

7.1 制御盤撤去/更新

  • 制御電源の3系統確認: AC100V/DC24V／UPS／非常電源／予備系電源。1系統見逃しで活線残
  • 盤内コンデンサ残留電荷: 平滑コンデンサ・サージアブソーバ。電源遮断後 5分以上の放電待機
  • 通信線抜線時のシステム影響: FF（Fieldbus）/PROFIBUS/Ethernet。抜いた瞬間に他制御系がフェールセーフ動作する可能性
  • 古い制御盤の有害物質: 1990年以前のトランス＝PCB含有疑い／盤内断熱材＝石綿含有疑い。事前分析必須
  • インタロック解除波及: 撤去対象盤が他設備のインタロック源になっている場合、解除が隣接プラント緊急停止を引き起こす

7.2 装置更新（圧力容器・反応器・熱交・ポンプ）

  • 残圧・残液・残熱・残ガス・残触媒の5残：縁切り・パージ・冷却・ベント・触媒抜出のシーケンス管理
  • MOC（変更管理）必須: 仕様変更（材質・容量・接続位置）は P&ID 更新と承認プロセス
  • 銘板重量と実重量の乖離: 残油・触媒残・ライニング・保温材で銘板＋20-50%重量増。クレーン能力余裕必須
  • 新設後の PSSR（Pre-Startup Safety Review）: 工事完了≠運転再開可。試運転前に安全点検必須
  • タイイン工事: 既設活管への接続。Hot Tap 採用時は専門業者・許可書必須

7.3 DCS更新（制御システム移行）

  • カットオーバー戦略の事前合意: Big Bang／Phased／Parallel Run。自社規模は Parallel Run 推奨（旧系統並行運転）
  • FAT/SAT: Factory（工場）/Site（現地）受入試験。ループチェック完了サイン（施工＋運転2名）
  • バックアップ＋ロールバック手順書: 切替失敗時に旧系統に戻せる手順。事前に書面化＋演習
  • I/O 仕様不一致リスク: アナログ4-20mA／デジタル接点／HART通信。旧新で電気特性・通信プロトコル差異がないか確認
  • 過渡期のオペレータ体制: 並行運転中はオペレータ負荷2倍。新人配置NG・熟練者ダブル配置
  • パラメータ移行: PID整定値・アラーム設定・トレンド設定。手作業移行はミス源 → スクリプト＋目視照合

7.4 モーター更新

  • 絶縁試験: 巻線対地 1MΩ以上（500V/1000V メガー）。試験中は 誘起高電圧 による感電危険
  • 芯出し: カップリング・プーリ。ダイヤルゲージ／レーザアライナで 0.05mm 以下。芯出し不良→異常振動・軸折損
  • 相回転確認: 結線後の試運転前に相回転計で正相確認。逆相起動でポンプ・送風機破損
  • インバータ容量整合: 既設インバータの容量が新モーター定格に対応するか（過電流保護設定見直し）
  • 重量物: 大型モーター（>500kg）は専用吊り具・タンデムリフト。銘板重量と実重量に注意

7.5 配管撤去/更新（タイイン含む）

  • 盲板（ブラインド）番号管理: 取付番号と撤去番号の照合表。「忘れ盲板」は復旧時の重大トラブル源
  • 残液・残圧確認の3点ルール: 上流弁閉→ベント開→圧力計0表示→低部ドレン排出
  • 断熱材撤去: 古い保温材（1980年代以前）はアスベスト含有率 > 0.1% の確率。事前分析・湿潤化・隔離養生
  • 切断方法選択: Hot Cutting（溶断）vs Cold Cutting（コールドソー）。可燃性ガス雰囲気は Cold 必須
  • タイイン時の隣接ライン誤開放: 「ブラインド対象が違った」事故は実在。現場照合＋運転側立会必須

7.6 ケーブル撤去/更新

  • 経路追跡の3手段: 図面（最新Rev確認）／タグ照合／信号注入器（パルス送信→受信機で特定）。図面のみは事故源
  • 既設活線混在の特定: ケーブルラック上で撤去対象と運転中が同居。誤切断＝感電・プラント停止
  • 通信線・制御線の他系統影響: 動力以外の信号線も「切れたら波及」リスク。シーケンス制御・インタロック信号の特定
  • 残留誘導電圧: 長尺ケーブル・他活線並走で誘起。接地→検電→接地短絡器具で安全確保
  • 外被材の有害物質: 古いケーブルの鉛シース・PCB含有油浸ケーブル。事前分析・特管処理

7.7 全案件共通の落とし穴（AI社員諮問・3者一致）

  1. 「書類のための書類」化: 立派なテンプレほど形骸化する。本テンプレは現場記入欄（測定値・写真・サイン・コメント）を強制することで防止
  2. 「過去にやったから大丈夫」のコピペ横展開: 自社連続プロセスは前回と残留物・運転条件が異なる。「前回工事との差分」欄を強制入力
  3. 「協力会社丸投げ・元請判子だけ」: 元請固有 hazard（運転側暗黙知）追記欄を最低1項目必須に
  4. 「停止＝ゼロエネルギー」過信: 全エネルギー源リスト記入欄で隠れエネルギー（UPS／残液／誘起電圧／残触媒）の見落とし防止
  5. 「図面と現場の不一致を現場が間違いと決めつけ」: 事前に正先合意欄を設けトラブル時の判断遅延防止
  6. 「MOC（変更管理）と RA の分離忘れ」: 工事RA で OK でも MOC 未取得で運転再開はコンプラ違反。MOC 番号入力欄で防止

8. 参照ガイドライン（国際・国内）

  • ISO 12100 / JIS B 9700: 機械安全 リスクアセスメント 3ステップメソッド
  • ISO 45001: 労働安全衛生マネジメントシステム
  • OSHA 29 CFR 1910.119: Process Safety Management (PSM) 14要素
  • CCPS Risk Based Process Safety: 20要素（米国化学工学会 CCPS）
  • IOGP Life-Saving Rules: 9項目（高所・墜落・LOTO・PTW・閉所・運転中作業・ライン断・吊り作業・運転中設備接近）
  • JCIA レスポンシブル・ケア: 日本化学工業協会 安全衛生・プロセス安全行動計画
  • JPCA 設備管理ガイドライン: 日本石油化学工業協会
  • HSE UK Asset Integrity: 英国安全衛生庁 設備健全性管理
  • KHK 高圧ガス保安協会 ガイドライン: 高圧ガス設備の保安・撤去基準
  • NFPA 70E / IEEE 1584: アークフラッシュ境界・PPE Category 算定（50V以上電気作業）
  • IEC 61511 SIL: Safety Instrumented System の機能安全（SIL1-4）・bypass運用
  • HAZOP（IEC 61882）: brownfield modification時の revalidation 必須
  • JNIOSH プロセス災害防止 RA マニュアル: 厚労省系・工事RAの4段階分離
  • 自社プラント「保安防災・労働安全衛生」: 重大事故定義（死1/重2 etc）・社内基準
