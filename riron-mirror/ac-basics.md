交流の電圧・電流は時間 t に対して正弦波で振動する。角周波数 ω は 1秒あたりに進む位相角（rad）＝ 振動の「角速度」：

 $\omega \;=\; 2\pi f \;=\; \dfrac{2\pi}{T}\quad[\mathrm{rad/s}]$

単位は rad/s（毎秒ラジアン）。周波数 f [Hz] が「1秒あたりの振動回数」、周期 T [s] が「1振動の時間」。三者は  $\omega = 2\pi f = 2\pi/T$  で1対1に対応する（例: f=50 Hz → ω≈314 rad/s）。

因果でつなぐと f が大きい → 単位時間の振動回数が多い → ω は大きい（位相が速く進む）。ω は 電気の角速度＝回転体の角速度の電気版で、フェーザー（位相子）として円運動として可視化される（v(t)=V_m sin(ωt+φ) は反時計回りに回る矢印の縦軸射影）。R/L/C の振る舞いを支配する量で、X_L = ωL ／ X_C = 1/(ωC) のように周波数依存性は全て ω 経由で現れる。🧩 チャンク階層 — どこを覚えているか自己確認第1層v(t) · V · Vm波形: V = V_m / √2 · 瞬時値 v(t) = V_m sin(ωt+φ)第2層ω · φ · フェーザー位相: 同周波数の波の角度ズレ・複素数で一本化第3層X_L · X_C · Z素子: X_L=ωL（L遅れ）/ X_C=1/(ωC)（C進み）/ Z=R+jX第1層 → 第2層 → 第3層 の順で迷子を自己診断。「Z 合成で詰まる」なら第3層、「位相符号で詰まる」なら第2層、「Vm vs V で詰まる」なら第1層から復習。💡⚡ 5秒で思い出す

## 3フェーザーと複素数（位相の数学）

正弦波  $v(t) = V_m \sin(\omega t + \varphi)$  は複素指数関数で表現できる：

 $v(t) = \text{Im}[V_m e^{j(\omega t + \varphi)}] = \text{Im}[\dot{V} e^{j\omega t}]$

ここで  $\dot{V} = V_m e^{j\varphi} = V_m \angle \varphi$  がフェーザー。定常状態では  $e^{j\omega t}$  は共通因子として消え、フェーザーだけで計算できる。微分 d/dt はフェーザー空間で jω 乗算 に対応 → 微分方程式が代数方程式に変換される。これが交流回路解析の本質。

## 4リアクタンスの周波数依存と位相（ELI/ICE の根拠）

インダクタ L: v = L (di/dt)。i = I_m sin(ωt) を代入 → v = ω L I_m cos(ωt) = ω L I_m sin(ωt + π/2)。電圧が電流より 90° 進む（ELI: E → L → I）。リアクタンスは  $X_L = \omega L$  → 周波数 f が高いほど X_L 大 → 高周波電流を妨げる。

コンデンサ C: i = C (dv/dt)。v = V_m sin(ωt) を代入 → i = ω C V_m cos(ωt) = ω C V_m sin(ωt + π/2)。電流が電圧より 90° 進む（ICE: I → C → E）。リアクタンスは  $X_C = 1/(\omega C)$  → 周波数 f が高いほど X_C 小 → 高周波電流を通しやすい。📍物理的直感

L は「電流の慣性」 → 急な変化を妨げる → 高周波ほど抵抗が大きい。C は「電荷の貯蔵」 → 充電・放電を繰り返す → 高周波ほど通しやすい（充放電が追従できる）。

> 図：ELI（インダクタ L）は電圧 v が電流 i より 90° 進む。ICE（コンデンサ C）は電流 i が電圧 v より 90° 進む。どちらも直角だが「誰が先か」が逆。「ELI the ICEman」で順序を固定。

## 5インピーダンス Z = R + jX

R と X（X_L or X_C）の直交合成。位相 φ は arctan(X/R) で実数部と虚数部の角度。

 $|Z| = \sqrt{R^2 + X^2}, \quad \varphi = \arctan(X/R)$

X が正（X_L）→ 誘導性（電流遅れ）／X が負（−X_C）→ 容量性（電流進み）。j（虚数）は「90° 進み」の演算子として捉えると複素数表現が腑に落ちる。

## 7引っかけポイント⚠勘違い①: コンセント 100V は最大値ではなく実効値
