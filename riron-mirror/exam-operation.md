## 1時間配分（90分・3パス＋見直し）

理論は計算量が多く時間との戦い。1問に粘りすぎると、確実に取れる問題を取りこぼす。「全問を順番に解く」のではなく、3回に分けて拾うのが原則。下表は目安（得意分野により前後する）。

| フェーズ | 時間 | やること |
| --- | --- | --- |
| 第1パス：A問題の即答拾い | 0〜35分 | 問1〜14を頭から。30秒で立式できる問題だけ即答。詰まったら印をつけて飛ばす（粘らない） |
| 第2パス：B問題 | 35〜65分 | 問15・16（必須）→ 問17・18は計算量の少ない方を選択。小問(a)→(b)は連鎖するので(a)を確実に取る |

## 2概算と選択肢消去

マークシートは答えそのものではなく「最も近い選択肢」を選べばよい。厳密計算で時間を使うより、概算で選択肢を絞る方が速いことが多い。

### 即値で覚えておく定数

| 値 | 概算 | 用途 |
| --- | --- | --- |
| √2 | ≈ 1.41 | 単相の実効値↔最大値 |
| √3 | ≈ 1.73 | 三相の線間↔相 |
| π | ≈ 3.14 | 角周波数 ω = 2πf |
| 力率 0.8 | sinφ = 0.6（3:4:5） | 有効↔無効電力 |
| 力率 0.6 | sinφ = 0.8 | 同上 |

### 消去の手順
- まず桁（×10ⁿ）を合わせる。選択肢が桁違いなら、概算だけで一発で絞れる
- 有効数字2桁で概算し、最も近い選択肢を選ぶ
- 係数（1/2・√3・2など）の付け忘れ／付けすぎを選択肢の間隔から逆算（例：選択肢が「ちょうど√3倍」並んでいたら、√3の扱いが論点）

## 3電卓の実戦テク（一般電卓のみ）⚠関数電卓は使えない理論は R²+X² の開平・√3倍 など平方根計算が頻出だが、使えるのは四則演算＋√キーの一般電卓のみ。普段から本番と同じ電卓に慣れておくこと。

| 機能 | 使いどころ |
| --- | --- |
| √キー | インピーダンス Z = √(R²+X²)、三相の √3。R²はR×=で出してから加算 |
| M+ / MR / MC（メモリ） | 途中結果を保持し、打ち直しを減らす。複数項の合計（例：P＝各相の和）で効く |
| ×= の連続乗算 | 二乗（R×=でR²）。x²キーがあればそれでも可 |
| GT（グランドトータル） | = を押すたびに加算される合計機能。直列の電圧降下合計などに |
💡押し間違い対策1計算ごとに概算と照合する癖をつける。桁が概算とずれたら、それ以上進まず即打ち直し。途中で電卓を信じ切らないことが、ケアレス失点を最も減らす。

## 4捨て問の判断（30秒ルール）

満点は不要、60点で合格。全問正解を狙うと時間切れになる。「解く問題」と「捨てる問題」を即断する基準を持つ。⚠30秒で飛ばす判断
- 30秒見て「何を求めるか」「最初の式」が立たない → 印をつけて飛ばす
- 計算が4段以上で見通せない → 後回し（第3パスで戻る）
- 典型的に重い論点：対称座標法・複雑な2次の過渡・見たことのない素子（→ 配点は他と同じ5点なので深追いしない）💡マークシートだから空欄厳禁無回答は0点だが、勘で1つ選んでも期待値は20%。捨て問でも、最後は消去法で2〜3択に絞ってから必ずマークを埋める。「分からない＝空欄」にしない。

## 5B問題の選び方（問17 vs 問18）

問17・18はどちらか1問だけを選んで解答する。選択を誤ると、解ける問題を自分で捨てることになる。
- 両方の小問(a)を30秒で読む。計算量が少ない／解法が見える方を選ぶ
- (a)が解けないと(b)も連鎖で落とす（(b)は(a)の結果を使うことが多い）。だから「(a)を確実に取れる方」を選ぶ
- 事前に得意分野で決め打ちする手もある（例：電子回路が得意なら問18がオペアンプ系のときは即決）📍B問題の詳しい攻略は別ページで小問(a)(b)の連鎖を含むB問題の解法トレーニングは、今後「B問題ストランド」ページで拡充予定。本セクションは本番での選択の即断に絞っている。

### 単位・次元で検算する（最強のミス検出）

答えの単位が、問われている量の単位と合わなければ、計算は必ずどこか間違っている。理論で最も確実な検算法。覚えた公式があやふやでも、単位が合う形を逆算すれば復元・検算できる。

| 量 | 単位 | 単位での検算例 |
| --- | --- | --- |
| 時定数 τ | s（秒） | τ=RC → [V/A]×[A·s/V]=[s] ／ τ=L/R → [V·s/A]÷[V/A]=[s] |
| 静電エネルギー | J | ½CV² → [A·s/V]×[V²]=[V·A·s]=[W·s]=[J] |
| 電力 P | W = V·A | VI ・ I²R ・ V²/R はすべて [W] になる |
| インピーダンス Z | Ω | √(R²+X²) も [Ω]（R・X と同じ次元どうしの和） |
💡使い方（3手順）
- 出した答えの単位が、問われた量の単位と一致するか確認する
- 式変形の各段で、両辺の次元（単位）が保存されているか見る
- 公式があやふやなら、単位が合う形を逆算して式を復元する

## 7図・ベクトル図・波形の読み取り（出たら何を見るか）

理論は図が前提の問題が多い。図が出たら、いきなり計算せず 「軸 → 基準 → 大きさの種類」 の順で読む手順を固定する。

| 手順 | 見るもの |
| --- | --- |
| ① 軸を読む | 縦軸・横軸が何の量で単位は何か |
| ② 基準を探す | ベクトル図＝基準ベクトル（ふつう電圧）／波形＝位相の基準となる波（ふつう電圧）／特性曲線＝原点・動作点 |
| ③ 大きさの種類 | 実効値か最大値か（取り違えが頻出） |

### ベクトル図（フェーザ図）
- フェーザは反時計回りが回転の正方向。基準（ふつう電圧）から見て、電流が反時計回り側にあれば進み＝容量性、時計回り側なら遅れ＝誘導性。
- 電圧と電流のなす角 φ が力率角で cosφ＝力率。ただし cosφ の値だけでは進み/遅れを区別できない → 「0.8（遅れ）」のように併記する。
- 合成は平行四辺形（または直交成分の和）で求める。

### 波形（時間波形）
- 縦軸＝瞬時値、横軸＝時間（or 角度 ωt）。中心線から山頂まで＝最大値 Vm（山〜谷の全幅は 2Vm）。
- 実効値＝Vm/√2 は正弦波のときだけ。方形波・三角波・整流波形では係数が異なる（波形率・波高率で換算）。
- 2波形のずれ＝位相差。周期 T、f＝1/T。進んでいる波ほど山が左に来る（フェーザの反時計回り＝進み と対応）。

### 特性曲線
- 共振カーブ：直列共振は f₀ でインピーダンス最小・電流最大／並列共振はインピーダンス最大・電源から見た線電流が最小（LC枝内を循環する電流はむしろ大きい）。
- トランジスタ出力特性（IC–VCE）、ダイオード V–I 特性 など：軸の量と動作点を最初に確認する。⚠向きの取り違えが力率問題の典型失点進み/遅れは基準（電圧）から見て、反時計回り＝進み（容量性）・時計回り＝遅れ（誘導性）。時間波形では進み＝山が左。この向きを逆にすると力率・位相の問題を落とす。
