 $Z \;=\; R + j\!\left(\omega L - \dfrac{1}{\omega C}\right)\;=\;R + jX\quad[\Omega]$

単位は Ω（オーム）。大きさは  $|Z|=\sqrt{R^2+X^2}$ 、位相は  $\tan\varphi = X/R$ 。実部 R は 抵抗成分（エネルギー消費）、虚部 X は リアクタンス成分（エネルギー一時蓄積）。

## 1原理：リアクタンス相殺としての共振🧠 脳に残す一言共振 ＝ X_L と X_C が完全に打ち消し合う特別な周波数。直列なら抵抗 R だけが残り Z 最小・電流最大。並列なら L・C 間で循環電流が閉じ電源から見た Z 最大・電流最小。🔑直感アナロジー⚙️RLC は バネ（C）・質量（L）・ダンパー（R）の機械振動と数学的に同型。共振 ＝ ブランコが最大に揺れる状態。固有振動数 f₀ で押すと、エネルギーが L と C の間を行き来し、R だけが正味の消費源として残る。

X_L = ωL は周波数 f に比例し、X_C = 1/(ωC) は周波数 f に反比例する。両者が交わる唯一の周波数が f₀ = 1/(2π√LC)。L と C だけで決まり R は f₀ に影響しない（R は Q＝共振の鋭さに影響）。

## 3共振条件 X_L = X_C の導出

直列RLC回路の複素インピーダンス：

 $\dot{Z} = R + j(X_L - X_C) = R + j\left(\omega L - \frac{1}{\omega C}\right)$

共振 ＝ 虚部がゼロになる周波数。すなわち：

 $\omega L = \frac{1}{\omega C} \quad \Rightarrow \quad X_L = X_C$

この条件下では Z = R（純抵抗的）となり、電流は電圧と同位相、大きさは  $I = V_s/R$ （直列共振の最大電流）。💡共振の定義は「虚部ゼロ」

共振の本質は「インピーダンスが純抵抗的になる周波数」。直列でも並列でも、共通の定義は虚部 = 0。そこから直列なら Z 最小、並列なら Z 最大が導かれる。

> 図：直列RLCの周波数特性。XL=ωL は右上がり、XC=1/(ωC) は右下がりで、両者が一致する周波数が共振 f0。このとき虚部が打ち消え、合成インピーダンス Z=√(R²+(XL−XC)²) は最小値 R になり電流が最大になる。「共振＝何かが最大」ではなく「Z 最小・I 最大」。

## 4共振周波数 f₀ の単位と桁感覚

共振条件 ωL = 1/(ωC) を ω について解くと：

 $\omega_0 = \frac{1}{\sqrt{LC}} \quad [\text{rad/s}], \qquad f_0 = \frac{1}{2\pi\sqrt{LC}} \quad [\text{Hz}]$

| L | C | ω₀ [rad/s] | f₀ [Hz] |
| --- | --- | --- | --- |
| 10 mH | 10 μF | 10⁴ | ≈ 1591 Hz |
| 10 mH | 1 μF | 10⁴·√10 ≈ 3.16×10⁴ | ≈ 5033 Hz |
| 1 H | 1 μF | 10³ | ≈ 159 Hz |
⚠単位変換は代入前に揃える

1 mH = 10⁻³ H、1 μF = 10⁻⁶ F。L = 10 mH、C = 10 μF なら LC = 10⁻² × 10⁻⁵ = 10⁻⁷... 失敗例だが、正確には L=10⁻² H・C=10⁻⁵ F なら LC=10⁻⁷ ⇒ √(LC)≈3.16×10⁻⁴ ⇒ ω₀≈3162 rad/s。代入前に SI 単位に正規化するクセをつけると桁ミスが消える。

## 5直列共振 vs 並列共振（完全比較）

電験3種では「直列か並列か」を最初に判別することが最重要。同じ「共振」でも振る舞いが正反対。

| 項目 | 直列共振 | 並列共振 |
| --- | --- | --- |
| インピーダンス | Z 最小（= R） | Z 最大（≈ R or L/Cr） |
| 電源から流れる電流 | 最大（I = V_s/R） | 最小 |
| 位相 | 電圧と電流が同位相 | 電圧と電流が同位相 |
| L・C の電圧/電流 | V_L = V_C = QV_s（電圧拡大） | L・C 間で大きな循環電流 |
| Q値 | Q = ω₀L/R | Q = R/(ω₀L) |
| 用途 | 選択フィルタ・受信機チューニング | タンク回路・発振器 |
| 主な危険 | L・C の過電圧（QV_s） | 循環電流による素子発熱 |

### 並列RLC：共振時の等価抵抗（R06上 問13 頻出）

並列回路はアドミタンス  $Y = G + jB$  で考える方が自然：

 $Y = \frac{1}{R} + j\left(\omega C - \frac{1}{\omega L}\right)$

共振条件は  $B = 0$ 、つまり  $\omega C = 1/(\omega L)$ （直列と同じ ω₀ = 1/√(LC)）。共振時は  $Y = G = 1/R$  だけ残るので  $Z_{\text{共振}} = R$ （最大）。

コイルに巻線抵抗 r が直列に入っている実回路（コイルが r + jωL）では、高Q近似で：

 $Z_{\text{共振}} \approx \frac{L}{Cr}$

r が小さいほど Z が大きく、電源電流が最小。この L/(Cr) は電力問題でも頻出。

### 直列共振時の各素子電圧（R04上 問9 頻出）

共振時 X_L = X_C ⇒ I = V_s/R（最大）。このとき：

 $V_R = IR = V_s, \quad V_L = IX_L = \frac{V_s}{R}\cdot\omega_0 L = Q\,V_s$

よって  $V_L/V_R = Q$ 。コイルと抵抗の電圧比がそのまま Q 値。V_C も同じ大きさだが位相が V_L と逆なので合計 V_L + V_C は実部上ゼロ（直列合計が V_s に等しく整合）。⚠共振時でも V_L・V_C は消えていない

大きさは V_L = V_C = QV_s（電源の Q 倍）。位相が逆なので回路全体としては純抵抗的に見えるだけで、各素子には Q 倍の電圧がかかっている。Q=10 なら入力 100V に対して 1000V。高Q電力回路では過電圧破損リスクがある。

> 図：直列共振時の電圧ベクトル。VL（上）と VC（下）は大きさが等しく（Q·Vs）位相が逆なので相殺し、回路全体では VR=Vs だけが見える。だが各素子には実際に Q 倍の電圧がかかる（高 Q で過電圧破損リスク）。

## 6Q値・帯域幅・選択度

Q値は共振の鋭さを表す無次元量。物理的には「1周期あたりの蓄積エネルギー / 損失エネルギー」の比率。

 $Q = 2\pi \cdot \frac{\text{1周期あたりの蓄積エネルギー}}{\text{1周期あたりの損失エネルギー}}$

| 回路 | Q の式 | R が大きくなると Q は |
| --- | --- | --- |
| 直列RLC | $Q = \omega_0 L / R = 1/(\omega_0 CR)$ | 下がる（損失増） |
| 並列RLC | $Q = R/(\omega_0 L) = \omega_0 CR$ | 上がる（並列損失減） |

帯域幅 BW（共振点で電力が半分になる周波数幅・−3dB 帯域）：

 $BW = f_0/Q \quad [\text{Hz}]$

Q が大きい ⇒ BW が狭い ⇒ 選択度が高い。ラジオの選局のように特定周波数だけ抜き出す用途では Q を大きくする。電力フィルタでは Q を程よく設定し過電圧と帯域を両立させる。💡🧐 Q値の4視点まとめ（監修者 input 反映「Q値とは何か」・2026-05-29 追加）

監修者さん指摘 (PR #85 同型): 「Q値の定義式は知っているが本質理解未到達」。Q値は同じ無次元量を 4 つの異なる側面で表現している。1つでも腑に落ちれば全て繋がる。

| 視点 | 表現 | 直感イメージ |
| --- | --- | --- |
| ① 共振の鋭さ | $Q = f_0 / BW$ | 共振曲線の山が高く幅が狭い ⇒ Q 大。ラジオの選局精度 |
| ② 電圧拡大率 (直列) | $Q = V_L / V_s = V_C / V_s$ | 共振時、L と C に印加電圧の Q 倍が現れる。Q=10 → 100V 入力で 1000V |
| ③ 蓄積/損失エネルギー比 | $Q = 2\pi \cdot \frac{W_{\text{蓄積}}}{W_{\text{損失}}}$ | L と C 間を行き来するエネルギーが、R で消費される量より何倍多いか |
| ④ 振動の持続性 | Q 周期で振幅 1/e に減衰 | 鐘・ピアノ弦が長く鳴る ⇔ Q 大。電気でも同じ |

本質を 1 行で: Q値 ＝ 「共振状態の良さの度合い」。鋭さ・拡大率・エネルギー比・持続性のどれで測っても同じ値。直列  $Q = \omega_0 L/R$ 、並列  $Q = R/(\omega_0 L)$  は「R の影響を直列と並列で逆方向に効かせる」だけ（直列は R 大で損失増 → Q 下がる、並列は R 大で電流制限 → Q 上がる）。

※ 監修者さんの暗記回避: 公式そのものより「4視点はすべて同じ Q を別角度で測っている」を脳に刷り込む。試験問題は4視点のどれかで聞いてくる。

## 7よくある勘違い（引っかけ集）⚠① 直列インピーダンスをスカラー和で計算

❌ Z = R + X_L + X_C と単純加算。✅ R とリアクタンスは直交ベクトル。Z = √(R² + (X_L − X_C)²)（直角三角形で考える）。⚠② 並列共振でも直列と同じ Q = ω₀L/R を使う（パターン28）

❌ 並列共振でも Q = ω₀L/R と書く。✅ 並列共振では Q = R/(ω₀L)（直列と逆数の関係）。並列回路では R が大きいほど電流を制限して Q が上がる。問題文で「並列」とあれば必ず確認する。（H26問10・R04上問10 類出）⚠③ 直列共振で Z が「最大」と誤解（パターン27）

❌ 「打ち消し合うから Z 最大」。✅ 打ち消し合い ⇒ リアクタンス消失 ⇒ Z = R（最小） ⇒ I 最大。「共振 ＝ 打ち消し合い ＝ 最小インピーダンス ＝ 最大電流」の因果を押さえる。（H24問7・R02問9 類出）⚠④ 並列共振で Z が「最小」と誤解（パターン28）

❌ 直列と同じく Z 最小・電流最大。✅ 並列共振は Y 最小 ⇒ Z 最大 ⇒ 電源電流最小。ただし L・C 間では大きな循環電流が流れている。⚠⑤ RC 回路に共振があると思う

❌ 「リアクタンスがあるから共振する」。✅ 共振は L と C の両方がないと起きない。RC 回路は X_C = 1/(ωC) の周波数応答（電流が周波数で変わる）のみ。R06下 問9 の論点はこの違い。⚠⑥ 共振時に V_L・V_C が「消える」と誤解

❌ 「打ち消し合うから両方ゼロ」。✅ 大きさは V_L = V_C = QV_s（電源の Q 倍）。位相が逆なので合計だけがゼロになる。高Q では実装上の過電圧リスクあり。

## 8拡張問い：電力品質とフィルタ応用📍🧐 RLC共振の現場応用（diffuse mode 起動）

電力系統の高調波フィルタは RLC 共振の直接応用。工場のインバータが発生する第5・第7高調波を、LC を選んで f₀ をその周波数に合わせて選択的に吸収する。タンク回路（並列共振）は逆に「特定周波数だけ通さない（ブロック）」用途で使われる。機械振動との対応（暗記不要の物理アナロジー）
- L ↔ 質量 m（運動エネルギー＝磁気エネルギー）
- C ↔ バネ定数の逆数 1/k（位置エネルギー＝静電エネルギー）
- R ↔ ダンパー（散逸＝抵抗損失）
- 共振角周波数 ω₀ = √(k/m) ↔ ω₀ = 1/√(LC)
- Q値 ＝ 振動が何周期続くかの目安（Q=10 なら約10周期で振幅 1/e）

機械系では Q 大 ＝ 鐘・ピアノ弦のように長く鳴り続ける。電気系の Q 大 ＝ 選択度が鋭いラジオフィルタ。同じ物理を別の道具で見ているだけ。🔬 Level 2：共振と微分方程式（試験範囲外・参考）

直列RLC回路の微分方程式： $L\frac{d^2i}{dt^2} + R\frac{di}{dt} + \frac{1}{C}i = \frac{dv}{dt}$ 。特性方程式の解は減衰係数  $\alpha = R/(2L)$  と共振角周波数  $\omega_0 = 1/\sqrt{LC}$  で決まる。
- α < ω₀（不足制動）：振動しながら収束
- α = ω₀（臨界制動）：最速収束、振動なし
- α > ω₀（過制動）：ゆっくり収束

Q値は  $Q = \omega_0/(2\alpha) = \omega_0 L/R$ 。Q 大 ＝ 共振が鋭く、過渡振動が長く続く。← 前2.3 交流電力次 →2.5 ブリッジ回路

- V_R = V_s に気づかない：共振時 I=V_s/R なので V_R=IR=V_s（電源電圧そのもの）
- X_L と X_C を別々に使う：共振では X_L=X_C=100 Ω。どちらで比をとっても Q=20
