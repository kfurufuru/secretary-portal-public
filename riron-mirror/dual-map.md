## 2双対が"裏切る"3つの非対称（要注意）

双対は美しいが、機械的に当てはめると失点する場面がある。得点を落とすのはたいていこの3点。⚠機械適用すると間違える3点
- 直列↔並列では R↔G の読み替えが必須。「並列でも R を足す」は誤り。直列は R の和、並列は G（=1/R）の和。読み替えを飛ばすと逆に間違える。
- C と L は直並列合成の向きが逆。コンデンサは直列で容量が減り・並列で増える（抵抗と逆）。コイルは抵抗と同じ（直列で増える）。双対の対称美に引きずられて向きを混同しない。
- X と B は単純な逆数ではない。Y=1/Z で実部・虚部が混ざるため、一般に B ≠ −1/X（純リアクタンスのときだけ B = −1/X）。
