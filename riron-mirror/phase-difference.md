## 0.このページ（位相差）を森・ミクロ・マクロで見る

同じテーマを 森（俯瞰）→ ミクロ（なぜ）→ マクロ（解く） の3層で見る。まず森でこのページの位置を掴み、ミクロで機構を、マクロで解き方を押さえる。

> 森（俯瞰）：交流の基礎（正弦波・ωt）と弧度法（rad）を土台に、位相差は 2 つの正弦波の時間ずれを角度で表す物差し。ここからコイル・コンデンサの応答、RLC 共振、力率 cosφ へ枝分かれする。

| 視点 | 何を見るか | 代表する式・量 |
| --- | --- | --- |
| ミクロ（なぜ） | 電圧と電流は微分でつながる。コイルでは電圧が電流の「傾き」になるため、電流が後追い＝π/2 遅れる（↓図解） | $v_L = L\,di/dt$ ・ $i_C = C\,dv/dt$ |
| マクロ（解く） | 波形は追わず、引数の差 φ と位相子（回転ベクトル）の角度差で扱う。試験はここ | $v = V_m\sin(\omega t + \phi)$ ・ $\phi = \arctan(\omega L/R)$ |
| 実務（意味づけ） | 設備で何に化けるか。力率 cosφ（進相コンデンサによる力率改善）、三相交流の 2π/3 位相差 | 力率 cosφ・三相 2π/3 rad |

▼ ミクロを図で掘り下げる：なぜコイルで電流が π/2 遅れるか

> ミクロ（なぜ）：理想コイル L・定常正弦波に固定（縦軸は i・v を各振幅で正規化）。v ＝ L·di/dt なので電圧は電流の「傾き」に比例し、i＝0 で傾き最大の瞬間に v が最大。結果、v は i より π/2 早くピークに達する＝電流は電圧より π/2 遅れる。コンデンサは i ＝ C·dv/dt で立場が逆になり、電流が π/2 進む。🔬 ミクロの因果（なぜ π/2 ずれるか）
- コイルに正弦波電流 i ＝ Iₘ sin ωt が流れる（理想コイル・定常状態）
- コイルの電圧は電流の変化率（傾き）に比例： $v_L = L\,di/dt$
- i ＝ 0 の瞬間に傾きが最大 → そこで v が最大 ⇒  $v = \omega L I_m \sin(\omega t + \pi/2)$
- 結果、電圧が π/2 先行 ＝ 電流は電圧より π/2 遅れ（ELI）。コンデンサは  $i_C = C\,dv/dt$  で逆＝電流が π/2 進み（ICE）

▶ マクロ（解く）の図解は、この下のセクション1「原理（時間ずれを角度で測る）」にある位相子（回転ベクトル）と時間波形の図を参照。

## 1.原理（時間ずれを角度で測る）

同じ周波数の 2 つの正弦波  $v_1 = V_m \sin\omega t$ 、 $v_2 = V_m \sin(\omega t + \phi)$  の 引数の差  $\phi$  が位相差。 $\phi > 0$  なら  $v_2$  は  $v_1$  より 「進む」（先にピークが来る）、 $\phi < 0$  なら 「遅れる」。5秒で思い出す
コイル L：電流は電圧より  $\pi/2$  遅れる（電流が後追い）。
コンデンサ C：電流は電圧より  $\pi/2$  進む（電流が先行）。
抵抗 R：電圧と電流は同位相（ずれゼロ）。

「進み・遅れ」の主語は通常 電流（i）が電圧（v）に対してどうか。混同しやすいので毎回主語を確認すること。e₁ ＝ Eₘ sin ωt（基準）e₂ ＝ Eₘ sin(ωt + θ₂) — θ₂ 進みe₃ ＝ Eₘ sin(ωt − θ₃) — θ₃ 遅れ

> 左：位相子（回転ベクトル） 反時計回りに回転し、各ベクトルの角度差が位相差。
右：時間波形 同じ角度差が「ピーク位置の時間差」として現れる（左へずれる＝進み、右へずれる＝遅れ）。観覧車アナロジー：A・B・C さんが等速で回る観覧車に乗っている。B を基準にすれば、進行方向側にいる A は「進み」、後ろ側の C は「遅れ」。基準を A に変えれば B は「遅れ」、C はさらに「遅れ」。
つまり「位相差」は2つのベクトル（ゴンドラ）間の角度差で、基準を変えれば符号も値も変わる。試験問題では必ず「何を基準にしているか」を最初に確認すること。

具体的な素子の例：抵抗 R は θ＝0、コイル L は  $\theta = -\pi/2$ （電流が電圧より遅れ）、コンデンサ C は  $\theta = +\pi/2$ （電流が電圧より進み）— 次のセクションで公式と一緒に整理する。

## 2.公式

| 素子 | 電圧 | 電流 | 位相差 |
| --- | --- | --- | --- |
| 抵抗 R | $v = V_m\sin\omega t$ | $i = (V_m/R)\sin\omega t$ | 0（同位相） |
| インダクタ L | $v = V_m\sin\omega t$ | $i = (V_m/\omega L)\sin(\omega t - \pi/2)$ | 電流が $\pi/2$ 遅れ |
| コンデンサ C | $v = V_m\sin\omega t$ | $i = \omega C V_m \sin(\omega t + \pi/2)$ | 電流が $\pi/2$ 進み |
| RL 直列 | 合成インピーダンス $Z = R + j\omega L$ | $\phi = \arctan(\omega L/R)$ （遅れ） |  |
| RC 直列 | 合成インピーダンス $Z = R - j/(\omega C)$ | $\phi = \arctan(1/(\omega CR))$ （進み） |  |

## 3.覚え方「エリの法則」ELI the ICE man（語呂ではなく順序の図像）
・ELI：L 回路では E（電圧）が I（電流）より先 → 電流は遅れ
・ICE：C 回路では I（電流）が E（電圧）より先 → 電流は進み
順序が「E L I」または「I C E」と並んでいる側が時間的に先。

注意：これは試験で素早く位相方向を思い出すための補助。公式根拠は  $v_L = L\,di/dt$ 、 $i_C = C\,dv/dt$  の微分関係であり、暗記より導出を優先する。

## 4.例題

問題:  $R = 30\,\Omega$ 、 $X_L = 40\,\Omega$  の RL 直列回路に交流電圧  $V = 100$  V を加えた。電流の位相は電圧に対し何度ずれるか。

解答:  $\phi = \arctan(X_L/R) = \arctan(40/30) \approx 53.1^\circ$  遅れ。
検算： $|Z| = \sqrt{30^2 + 40^2} = 50$  Ω、 $I = 100/50 = 2$  A。
