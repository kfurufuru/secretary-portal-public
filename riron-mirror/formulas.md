# D 公式集

「全部暗記」は非効率。出発点の最小セットだけ覚え、残りは導出する。直前期は下の「まず覚える」表だけ総ざらいすればよい。下の早見表（1〜4節）はその確認用。

## ★1 まず覚える最小公式（出発点・直前期はここだけ）

| 公式 | なぜ暗記か |
| --- | --- |
| F = kQ₁Q₂/r²（クーロン） | 静電気の出発点 |
| C = εS/d | 構造で決まる定義 |
| e = −N dΦ/dt（ファラデー） | 電磁誘導の出発点 |
| XL = ωL ・ XC = 1/(ωC) | リアクタンスの定義 |
| V = IR（オーム） | 回路の出発点 |
| 実効値 = 最大値 / √2（正弦波） | 正弦波の定義値（方形波は実効値＝最大値で別） |
| P = VIcosφ ／ 三相 P = √3·VL·IL·cosφ | 電力の基本形。三相は線間電圧 VL・線電流 IL（相値と混同しない） |
| f₀ = 1/(2π√(LC))（共振） | 完成形を暗記（導出の核は XL = XC が共振条件） |
| W = ½CV²（主）／ ½LI² は双対 | エネルギーの出発点（C↔L・V↔I で対） |
| τ = RC ／ τ = L/R | 時定数。RL は L/R（R が分母）＝双対でも逆数構造なので両方暗記 |
| 反転 Av = −Rf/Ri ／ 非反転 Av = 1 + Rf/Ri | 符号(−)と(+1)の有無を込みで暗記 |

## ★2 覚えなくてよい公式（出発点から導ける）

| 導出できる公式 | 出発点 → 導き方 |
| --- | --- |
| 直列/並列の合成（抵抗・容量） | オームの法則 ＋ KVL/KCL |
| Z = √(R²+(XL−XC)²) | R と X を直交ベクトル合成（三平方の定理） |
| Pmax = Vth²/(4Rth)（Rth=RL で最大） | I=Vth/(Rth+RL) を代入 → P=Vth²·RL/(Rth+RL)² を RL で微分 → RL=Rth → Pmax=Vth²/(4Rth) |
| 無効電力 Q = VIsinφ ・ 皮相 S = VI | 電力三角形（P = VIcosφ の対） |
| ½LI² | ½CV² の双対（C↔L・V↔I）→ インダクタンス |
| 全波整流の平均 2Vm/π（半波は Vm/π） | 正弦波の半周期を積分 |
導出の落とし穴（双対の機械適用に注意）
「双対だから」と当てはめて間違える典型 → τ は RL では L/R（R×L ではない・次元も不一致）、並列の合成は R ではなく G（=1/R）で足す。覚えるより導く方が速い場面でも、この2点だけは向きを取り違えやすい。
