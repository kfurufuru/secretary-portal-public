▼ ミクロを図で掘り下げる：なぜ平衡だと検流計に電流が流れないか

> ミクロ（なぜ）：ブリッジは左右2つの分圧器。R₁R₄ = R₂R₃ のとき分圧比が等しく中点電位が一致（Vc = Vd）し、検流計の両端に電位差がかからないので Ig = 0。ゼロになるのは橋絡部だけで、4辺には電流が流れ続けている。🔬 ミクロの因果（なぜ平衡で Ig = 0 か）
- 電源 E が左右2本の直列路（R₁–R₃ と R₂–R₄）に同じ電圧をかける
- 各路は分圧器：中点の電位は  $V_C = E\,\dfrac{R_3}{R_1+R_3}$ ・ $V_D = E\,\dfrac{R_4}{R_2+R_4}$
- R₁R₄ = R₂R₃ のとき分圧比が等しく Vc = Vd → 検流計両端の電位差がゼロ
- 結果、橋絡部だけ Ig = 0（平衡）。4辺には電流が流れ続ける → 平衡条件から  $R_x = \dfrac{R_2}{R_1}R_3$  で未知抵抗が測れる

▶ マクロ（解く）の図解は、この下の図①（ひし形と対辺の積）・図②（橋絡辺の見抜き方）と、解説タブ4節の図③（不平衡ブリッジのテブナン化）を参照。⚠🎯 まずこれ：ブリッジは「対辺の積」で平衡判定（ここから全部決まる）
- 向かい合う対辺の積を比べる：R₁·R₄ と R₂·R₃。等しければ平衡（検流計 Ig=0）。
- 平衡なら橋絡辺（検流計の辺）を取り除ける → 単純な直並列に落として解く。不平衡なら取り除けない → テブナンで解く。

※ 隣り合う辺の積（R₁·R₂＝R₃·R₄）ではないのが最頻ひっかけ（→ 解説タブ 6節）。平衡＝橋絡部だけ電流ゼロで、4辺には電流が流れている。

使う記号: R₁〜R₄（ブリッジ4辺の抵抗・Ω）／Rx（測定したい未知抵抗・Ω）／Rg（検流計の内部抵抗・Ω）／Ig（検流計に流れる電流・A）／E（電源電圧・V）／G（検流計＝橋絡部）／Vth（テブナン開放電圧・V）／Rth（テブナン等価抵抗・Ω）／L（インダクタンス・H）／C（静電容量・F）📝📘 ブリッジ回路 とは（定義）

4本の素子をひし形（◇）に組み、対角の一方に電源、もう一方に検流計を渡した回路。天秤のように左右の分圧バランスが取れると検流計に電流が流れない（平衡）。

平衡条件は「向かい合う対辺の積が等しい」R₁R₄ = R₂R₃。これを使えば既知素子から未知素子を精密に測れる（ホイートストンブリッジ＝抵抗測定）。不平衡時は検流計電流をテブナンの定理で求める。🧩 チャンク階層 — どこを覚えているか自己確認第1層平衡 / 不平衡対辺の積が等しい（R₁R₄=R₂R₃）＝平衡＝Ig=0／等しくない＝不平衡＝Ig≠0第2層橋絡辺の扱い平衡なら橋絡辺(検流計)は開放/短絡どちらでも同じ＝取り除いて直並列／不平衡なら取り除けない第3層解法選択平衡→Rx=R₂R₃/R₁（比例辺×標準辺）／不平衡→テブナン Ig=Vth/(Rth+Rg)第1層（平衡か）→ 第2層（橋絡辺を消せるか）→ 第3層（公式かテブナンか）。迷ったら第1層に戻り「対辺の積を比べる」。これで以降の解法がすべて決まる。💡⚡ 5秒で思い出す

ブリッジ ＝ 天秤。左皿と右皿の分圧が釣り合うと指針（検流計）がゼロ。釣り合いの式は対辺の積が等しい。
- 平衡条件：R₁·R₄ = R₂·R₃（向かい合う対辺の積・隣接辺ではない）
- 平衡時の検流計：Ig = 0（ただし4辺には電流が流れている）
- 未知抵抗：Rx = (R₂/R₁)·R₃（比例辺の比 × 標準辺）
- 橋絡辺の処理：平衡なら同電位 → 開放/短絡どちらでも同じ → 取り除ける
- 不平衡：平衡条件は使えない → テブナンで Ig = Vth/(Rth+Rg)

## 1原理（なぜ平衡で検流計に電流が流れないか）🔑直感アナロジー⚖️ブリッジ回路は天秤のイメージ。左皿（左の分圧点）と右皿（右の分圧点）の電位が等しくなると、橋を渡る指針（検流計）に電流が流れない。これが平衡。

検流計の両端（左ノード C・右ノード D）の電位差がゼロになるのが平衡。C は R₁・R₃ による分圧、D は R₂・R₄ による分圧で決まる。2つの分圧比が等しいとき VC=VD → 検流計に電位差がかからず Ig=0。📍平衡条件 R₁R₄=R₂R₃ の導出（暗記でなく分圧で出す）

左ノード C と右ノード D を、電源の下端（B＝基準0V）から見た分圧で表す（上辺＝R₁・R₂、下辺＝R₃・R₄）：

 $V_C = E\,\dfrac{R_3}{R_1+R_3}, \qquad V_D = E\,\dfrac{R_4}{R_2+R_4}$

平衡＝VC=VD とおいて E を消すと：

 $\dfrac{R_3}{R_1+R_3} = \dfrac{R_4}{R_2+R_4} \;\Rightarrow\; R_3(R_2+R_4)=R_4(R_1+R_3)$

両辺を展開すると R₃R₄ が相殺し、R₂R₃ = R₁R₄ ＝向かい合う対辺の積が等しい。分圧比 R₁/R₂=R₃/R₄ と書いても同じ。

## 2なぜブリッジで詰まるか（メタ認知）

ブリッジは「公式は知っているのに失点する」典型。原因は計算ミスではなく認知上の壁。症状と誤モデルを1表で対応付け、以降を読む前に「自分が今どこで詰まっているか」を意識する。

| 症状（つまずきポイント） | 誤モデル（脳内バグ） | 正モデル（突破の鍵） |
| --- | --- | --- |
| ① 平衡条件の辺を取り違える | 「隣り合う辺の積」R₁R₂=R₃R₄ だと思う | 向かい合う対辺の積 R₁R₄=R₂R₃（→ 1節・6節） |
| ② 平衡＝全電流ゼロと誤解 | 平衡なら回路に電流が流れないと思う | ゼロは橋絡部（検流計）だけ。4辺には電流が流れる（→ 3節） |
| ③ 不平衡に平衡条件を使う | ブリッジならいつでも対辺の積で解けると思う | 不平衡（R₁R₄≠R₂R₃）はテブナンで Ig（→ 4節） |
| ④ 橋絡辺の位置で覚える | 「縦（中央）の辺は電流が流れない」と図の向きで暗記 | 役割で覚える＝対角中点をつなぐ辺＝橋絡辺、平衡時のみゼロ（→ 3節） |
| ⑤ 交流ブリッジで位相を忘れる | インピーダンスの大きさだけ合わせて平衡と思う | 交流は複素数の積で実部・虚部とも一致が必要（→ 5節） |

## 3「平衡＝橋絡部だけ電流ゼロ」の正しい読み方

平衡で電流がゼロになるのは検流計（橋絡辺）だけ。電源から見れば4辺には電流が供給され続けている。「天秤の針がゼロでも、天秤全体は動いている」イメージ。⚠⚠️ よくある覚え違い：「縦（中央）の辺には電流が流れない」

橋絡辺（検流計の位置にある辺）に電流が流れないのは 図の向きとは無関係で、ブリッジが平衡しているときだけ。「縦の辺は流れない」と位置で覚えると、橋絡辺が斜めや横に描かれた瞬間に見抜けなくなる。覚えるべきは位置ではなく役割：「四角形の対角中点をつなぐ辺＝橋絡辺。平衡時のみ電流ゼロ」。💡取り除く前に必ず平衡チェック（直並列に落とす手順）

① 対辺の積を比べる（R₁R₄ = R₂R₃）→ ② 平衡なら橋絡辺の両端は同電位なので、その辺は開放しても短絡しても結果は同じ。取り除いて単純な直並列に落とす → ③ 不平衡なら取り除けない。テブナン・キルヒホッフ・Δ-Y変換で解く。

## 4不平衡ブリッジのテブナン解法

過去問で最もつまずく「不平衡ブリッジの検流計電流を求めよ」への対処。平衡条件は使えないので、検流計の位置でテブナン等価回路を作る。

> 図③：不平衡ブリッジは平衡条件が使えない。検流計を外して開放電圧 Vthを求め、電源を短絡して等価抵抗 Rth（R₁∥R₃ と R₂∥R₄ の直列）を求め、Ig = Vth/(Rth+Rg) で検流計電流を出す（テブナンの定理）。
- 検流計 Rg を外す（開放）：左右が2つの分圧回路になる。開放電圧  $V_{th}=V_C-V_D=E\left(\dfrac{R_3}{R_1+R_3}-\dfrac{R_4}{R_2+R_4}\right)$
- 電源を短絡：R₁とR₃が並列、R₂とR₄が並列になり、その2つが直列。 $R_{th}=\dfrac{R_1 R_3}{R_1+R_3}+\dfrac{R_2 R_4}{R_2+R_4}$
- 検流計電流： $I_g=\dfrac{V_{th}}{R_{th}+R_g}$ 📍平衡条件との関係（特殊ケース）

R₁R₄ = R₂R₃ のとき上の Vth の括弧の中がゼロになり Vth=0 → Ig=0。つまり平衡は「不平衡テブナン式で Vth=0 とおいた特殊ケース」。両者は別公式ではなく、同じ式の一部。

## 5交流ブリッジ（マクスウェル・ウィーン・シェーリング）

交流ブリッジは直流ブリッジの「抵抗」を「インピーダンス」に置き換えたもの。平衡条件の構造は同じ：向かい合う対辺のインピーダンスの積が等しい（Z₁Z₄ = Z₂Z₃）。ただし複素数なので実部・虚部の両方が一致して初めて平衡。

| ブリッジ | 測定対象 | 特徴・覚えどころ |
| --- | --- | --- |
| マクスウェル | インダクタンス L | 抵抗とコンデンサで L を測る（L=R₂R₃C）。ω を含まず周波数非依存で測れる |
| ウィーン | 周波数 f | CR素子で特定周波数のみ平衡 → 周波数測定／発振器に応用（f = 1/(2πRC)） |
| シェーリング | 静電容量・誘電損 | コンデンサの損失角 δ（tanδ）を測定。高電圧絶縁試験に使う |
📍交流ブリッジで検出器に「耳（ヘッドフォン）」を使う理由

交流ブリッジでは検出器にヘッドフォンを使う場合がある。平衡＝検出器の出力ゼロ＝音が聞こえなくなる瞬間（音消し法）。直流用の検流計は交流ブリッジに使えない（直流しか反応しない）点が選択肢の罠になる。

## 6引っかけポイント（対辺 vs 隣接辺）✅正解の原則ブリッジの平衡条件は「向かい合う対辺の積が等しい」。正しくは R₁·R₄ = R₂·R₃（上左×下右 = 上右×下左）。⚠よくある誤解上辺左R₁・上辺右R₂・下辺左R₃・下辺右R₄で、平衡条件を隣接辺の積 R₁·R₂ = R₃·R₄ と書いてしまう。🔍判別ステップ
- ブリッジを菱形（◇）で描き直し、電源の対角と検流計の対角を確認する
- 検流計が渡っている左右ノードに着目し、各ノードの分圧（R₁/R₃ 系・R₂/R₄ 系）を考える
- 平衡条件は対角どうしの積 R₁·R₄ = R₂·R₃（隣接辺ではない）と書く
- 迷ったら分圧比 R₁/R₂ = R₃/R₄ から積の形に変形して確認する出典: H19 問6・H18 問16 類出✅正解の原則平衡で電流がゼロになるのは検流計（橋絡辺）だけ。4辺には電流が流れており、電源から電流が供給されている。⚠よくある誤解「平衡＝回路全体の電流がゼロ」と誤解し、各辺の電流まで0として計算する。🔍判別ステップ
- 検流計（橋絡辺）と4辺の役割を区別する
- 平衡時は橋絡辺の両端が同電位 → 橋絡辺の電流のみ0
- 4辺は単純な直並列として電流が流れていることを確認する出典: bridge.md よくある勘違いTOP3-②（出典: 旧wiki監修）✅正解の原則不平衡（R₁R₄≠R₂R₃）では平衡条件は使えない。検流計電流はテブナンの定理で Ig=Vth/(Rth+Rg)。⚠よくある誤解不平衡なのに対辺の積（平衡条件）を立てて未知素子を求めようとする。🔍判別ステップ
- まず対辺の積を比べて平衡/不平衡を判定する
- 不平衡なら検流計を外して開放電圧 Vth を計算（左右の分圧の差）
- 電源を短絡して等価抵抗 Rth（R₁∥R₃ と R₂∥R₄ の直列）を計算
- Ig = Vth/(Rth+Rg) で求める（Rg は検流計抵抗）出典: R02 問7（直流）類題

## 1ブリッジ問題の解法フロー（汎用5ステップ）解き方・判断手順 （ブリッジ回路（平衡判定起点））
- 図を整える：菱形（◇）or「四角形＋対角線1本」に描き直し、橋絡辺（検流計の辺）を特定する
- 平衡判定：向かい合う対辺の積を比べる（R₁R₄ と R₂R₃）。等しい＝平衡／等しくない＝不平衡
- 平衡なら：Ig=0。未知抵抗 Rx=(R₂/R₁)R₃／合成抵抗を問うなら橋絡辺を取り除き直並列に落とす
- 不平衡なら：検流計を外して Vth（左右の分圧の差）→ 電源短絡で Rth（R₁∥R₃ + R₂∥R₄）→ Ig=Vth/(Rth+Rg)
- 検算：単位を Ω・V・A に揃え、対辺と隣接辺を取り違えていないか・橋絡辺の電流だけ0か最後に確認

## 2例題1 — 平衡条件から未知抵抗📊 このセクションの理解度:◎○△×📝例題1（計算・旧wiki監修問題）ホイートストンブリッジで R₁=100Ω、R₂=200Ω、R₃=150Ω のとき、平衡条件から未知抵抗 R₄ を求めよ。解答・解説

Step①：平衡条件は対辺の積 R₁·R₄ = R₂·R₃

Step②：R₄ = R₂·R₃ / R₁ = (200 × 150) / 100 = 30000 / 100

答え：R₄ = 300 Ω💡この例題のツボ対辺の積 R₁·R₄ = R₂·R₃。R₄ と組むのは対角の R₁（隣接の R₂・R₃ ではない）。隣接辺で立式すると R₄=75Ω 等の誤答になる。

## 3例題2 — 不平衡ブリッジの検流計電流（テブナン）📊 このセクションの理解度:◎○△×📝例題2（計算・代表例題）E=10V、R₁=10Ω、R₂=30Ω、R₃=40Ω、R₄=20Ω、検流計 Rg=20Ω のブリッジ。検流計に流れる電流 Ig を求めよ。（上辺＝R₁・R₂、下辺＝R₃・R₄、橋絡部に検流計）

> 図③：不平衡ブリッジは平衡条件が使えない。検流計を外して開放電圧 Vthを求め、電源を短絡して等価抵抗 Rth（R₁∥R₃ と R₂∥R₄ の直列）を求め、Ig = Vth/(Rth+Rg) で検流計電流を出す（テブナンの定理）。解答・解説

Step①：平衡判定 R₁·R₄=10×20=200、R₂·R₃=30×40=1200 → 不平衡（テブナンで解く）

Step②：開放電圧 Vth = VC − VD = 10×40/(10+40) − 10×20/(30+20) = 10×0.8 − 10×0.4 = 8 − 4 = 4V

Step③：電源短絡で Rth = (10×40)/(10+40) + (30×20)/(30+20) = 8 + 12 = 20Ω

Step④：Ig = Vth/(Rth+Rg) = 4/(20+20) = 4/40 = 0.1A

答え：Ig = 0.1 A（= 100 mA）

## 4不平衡ブリッジ 解法カード（5択・代表例題）📊 このセクションの理解度:◎○△×代表例題（不平衡ブリッジ型）問—

E=10V、上辺 R₁=10Ω・R₂=30Ω、下辺 R₃=40Ω・R₄=20Ω、橋絡部の検流計 Rg=20Ω。検流計に流れる電流 Ig はいくらか。
- ① 0 A
- ② 0.05 A
- ③ 0.1 A ← 正解
- ④ 0.2 A
- ⑤ 0.5 Aまず対辺の積で平衡/不平衡を判定する。R₁R₄=200・R₂R₃=1200 で不平衡 → テブナンで解く。解き方・判断手順 （不平衡ブリッジ（テブナン））
- 平衡判定：R₁R₄=200 ≠ R₂R₃=1200 → 不平衡（対辺の積が不一致）
- 検流計を外す → 開放電圧 Vth = 10×40/(10+40) − 10×20/(30+20) = 8 − 4 = 4V
- 電源短絡 → Rth = (10×40)/50 + (30×20)/50 = 8 + 12 = 20Ω
- Ig = Vth/(Rth+Rg) = 4/(20+20) = 0.1A
- 検算：平衡(Vth=0)ならIg=0のはず → 不平衡でVth=4V≠0と整合。単位 V/Ω=A を確認📝 まず自分で解答 → 👁 正解・解説を見る過去問は常に即表示にする⚠本問のひっかけポイント（5項目）
- 平衡判定を飛ばし、対辺の積で未知素子を出そうとする（本問は不平衡でテブナン）
- 開放電圧 Vth で左右の分圧の差を取らず、片方だけで答える
- 等価抵抗 Rth で R₁∥R₃ と R₂∥R₄ の直列合成を、並列と取り違える
- 検流計抵抗 Rg を分母に加え忘れる（4/20=0.2A の誤答が出る）
- Ω のまま vs S（コンダクタンス）で並列計算の式を取り違える
