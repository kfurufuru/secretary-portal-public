## 0このページ（トランジスタ）を森・ミクロ・マクロで見る

同じテーマを 森（俯瞰）→ ミクロ（なぜ）→ マクロ（解く） の3層で見る。まず森でこのページの位置を掴み、ミクロで機構を、マクロで解き方を押さえる。

> 森（俯瞰）：半導体で学ぶ pn接合・キャリアを土台に、トランジスタは小さな入力（BJT はベース電流 I_B・FET はゲート電圧 V_GS）で大きな電流を制御する素子。ここからオペアンプ・増幅/発振回路・スイッチング動作へ枝分かれする。

| 視点 | 何を見るか | 代表する式・量 |
| --- | --- | --- |
| ミクロ（なぜ） | エミッタから注入された電子が極薄のベースをほぼ素通りしてコレクタに達し、再結合したごく一部の補充分だけが I_B になる。だから小さな I_B で大きな I_C（↓図解） | 電子注入・薄いベース・再結合 |
| マクロ（解く） | 電子1個ずつは追わず、測れる量で扱う。試験はここ | $I_C = h_{FE} I_B$ ・ $I_E = I_B + I_C$ ・ $V_E = V_B - V_{BE}$ |
| 実務（意味づけ） | 設備で何に化けるか。センサ信号の増幅（熱電対→制御器）、ソレノイド駆動のスイッチ、MOSFETの高入力インピーダンスで信号源を乱さない | 増幅回路・ドライバ回路・4〜20mA送信器 |

▼ ミクロを図で掘り下げる：なぜ小さな I_B で大きな I_C を制御できるか

> ミクロ（なぜ）：NPN のアクティブ領域（V_BE ≈ 0.6 V で順バイアス・コレクタ接合は逆バイアス）では、エミッタから注入された電子の大部分が極薄のベースを素通りしてコレクタに達し、ベースで再結合したごく一部の補充分だけがベース電流になる。素通り : 再結合の比がほぼ一定（≈ h_FE : 1）なので I_C = h_FE·I_B が成り立つ。🔬 ミクロの因果（なぜ小さな I_B で大きな I_C か）
- V_BE ≈ 0.6 V の順バイアスで、エミッタ（N）から電子がベース（P）へ大量に注入される
- ベースは極めて薄いので、電子の大部分（約99%）は再結合せずコレクタ接合まで素通りする
- 逆バイアスのコレクタ接合の電界が電子を吸い込み、大きなコレクタ電流 I_C になる
- ベースで再結合したごく一部の補充分だけが I_B。素通り : 再結合 ≈ h_FE : 1 だから  $I_C = h_{FE} I_B$

▶ マクロ（解く）の図解は、解説タブ「直流動作点と電流帰還バイアスの『なぜ』」にある電流帰還バイアス回路図（分圧 → V_E = V_B − 0.6 → I_C ≈ V_E/R_E の解き筋がそのまま図になっている）を参照。

使う記号: IB（ベース電流）／IC（コレクタ電流）／IE（エミッタ電流）／hFE（直流電流増幅率・無次元）／VBE（ベース-エミッタ間電圧・Si≈0.6V）／VCE（コレクタ-エミッタ間電圧）／VCC（電源電圧）／RE（エミッタ抵抗）／RC（負荷抵抗）／VGS（ゲート-ソース電圧）／Vth（しきい値電圧）／PC（コレクタ損失・W）📝📘 直流電流増幅率 β（h_FE） とは（定義）

バイポーラトランジスタ（BJT）は 小さなベース電流 I_B で大きなコレクタ電流 I_C を制御する素子。その「何倍に増幅されるか」が β：

 $\beta \;=\; \dfrac{I_C}{I_B} \;(= h_{FE}), \qquad I_E = I_B + I_C = (1+\beta)\,I_B$

無次元（電流÷電流）で、実用素子では数十〜数百。h_FE はデータシート上の表記で中身は同じ。エミッタ電流 I_E は I_B と I_C の合流（KCL）なので 3電流のうち2つが分かれば残りが決まる。

因果でつなぐと I_B を少し増やす → β 倍の I_C が増える → 電流で電流を制御（増幅）。これが BJT＝電流制御の意味。対になる FET は ゲート電圧で制御（電圧制御・入力電流ほぼゼロ）で、この対比が電子回路の選択問題の出発点。🧩 チャンク階層 — どこを覚えているか自己確認第1層IC=hFE·IB増幅の基本: 小さなI_Bが大きなI_Cを制御第2層IE=IB+IC · VE=VB−VBE電流収支とバイアスの2式第3層IC≈VE/RE動作点: バイアス回路で Q点を確定第1層 → 第2層 → 第3層 の順で自己診断。「バイアス回路で詰まる」なら第3層、「I_E の電流収支で詰まる」なら第2層、「h_FE の意味で詰まる」なら第1層から復習。💡⚡ 5秒で思い出す

小さい入力で大きな電流を操るスイッチ兼増幅器。BJT＝電流制御・FET＝電圧制御の二択がすべての出発点。
- BJT：電流制御（IC=hFE·IB）・入力Z低・熱暴走に注意
- FET：電圧制御（VGSでチャネル開閉）・入力Z高・温度安定
- 電流収支：IE=IB+IC（全領域で常に成立）
- バイアス：VB分圧 → VE=VB−0.6 → IC≈VE/RE
- 3接地：エミッタ接地のみ位相反転（180°）

💡🔑 接合形FET（JFET）の動作原理 ― R4上 問11 で問われた中身

「Nチャネル」＝電流の通り道がN形という意味。BJT が層の並び（PNP/NPN）で命名するのに対し、JFET は通り道（チャネル）の材料で命名する。N形の棒の両脇を P形ゲートで挟み、ゲートを逆バイアス（VGS ≤ 0）にして使う。ゲートには電流が流れず、電圧で空乏層の厚み＝通り道の幅だけを変える ＝ 電圧制御。

> 図：Nチャネル接合形FET。ゲートの逆バイアスを強める（VGS を負に）ほど空乏層が拡大→Nチャネルが狭まる→ID 減少。両側の空乏層がくっつくとピンチオフ（ID≒0）。MOSFETのエンハンスメント型（VGS>Vth でON）とは動作方向が逆。

| R4上 問11 の空欄 | 正答 | 理由 |
| --- | --- | --- |
| (ア) チャネルを流れるキャリヤ | 多数キャリヤ | N形の多数キャリヤ＝電子が S→D を流れる |
| (イ) 逆電圧で n チャネルの幅は | 減少 | 空乏層がチャネルに食い込み通り道が狭まる |
| (ウ) ドレーン電流 I_D は | 減少 | 通り道が狭まり抵抗増→電流減（V_P で≒0） |
| (エ) FET は | 電圧制御形 | ゲート電流ほぼ0・V_GS だけで I_D を調整 |
📝nチャネル vs pチャネル／空乏型 vs エンハンスメント型（混同注意）nチャネル：電子（多数キャリヤ）が通り道を流れる。pチャネル：正孔が流れる（電圧の向きが逆）。
JFET（空乏型）：最初から通り道があり、nチャネルは VGS ≤ 0 で使う。VGS=0 で最大・負にするほど ID 減少（R4上 問11）。
エンハンスメント型MOSFET：最初は通り道が無く、nチャネルは VGS > Vth で初めてチャネル形成＝ON（R05下・H23）。
⚠️ JFET と エンハンスメントMOSFET は動作方向が逆。「Vth でON」は MOSFET 専用で、JFET には当てはめない。

> 図：JFET の N／P 判別。回路記号のゲートの矢印が内向き（チャネルを指す）＝Nチャネル形、外向き＝Pチャネル形。Nは多数キャリアが電子・ドレーンが＋・ゲートがP形で、Pはすべて逆。上の空乏層の図はこのNチャネル形の動作。

## 6増幅回路の利得・帰還・発振（問13・問18対策）💡⚡ 問13・問18 の計算はここ

問11 が BJT／FET の論説中心なのに対し、問13・問18 は増幅度・帰還・発振の計算。エミッタ接地利得・FETソース接地・負帰還・発振の4系統を押さえれば計算枠を取り切れる。レイヤーB — 応用公式

| --- | --- | --- | --- |
| $A_v \approx -\,h_{FE}\,\dfrac{R_C}{h_{ie}}$ | エミッタ接地の電圧利得（マイナス＝入力と逆位相）。h_ie は入力インピーダンス | 小信号等価回路での近似。エミッタ接地増幅回路 | 大信号・非線形領域では不可。電流利得 h_FE と混同しない（別の式） |

| $A_f = \dfrac{A}{1 + A\beta_f} \;\to\; \dfrac{1}{\beta_f}$ | 負帰還の閉ループ利得（A=開ループ利得・β_f=帰還率）。深い帰還 Aβ_f≫1 では 1/β_f に収束 | 負帰還増幅。利得を下げて安定性・帯域を得る | 正帰還（位相0°で Aβ_f=1）になると発振側の扱い |
📝コレクタ接地（エミッタフォロワ）の直流動作点 — R07上問18・H30問16

出力をエミッタから取り出す。電圧利得≈1・電流利得大・入力Z高＝バッファ（インピーダンス変換）。計算上の要はコレクタ側に負荷抵抗が無いこと。
- V_BE ≈ 0.6V（Si）
- ベース電圧（分圧） $V_B = V_{CC}\dfrac{R_2}{R_1+R_2}$
-  $V_E = V_B - V_{BE}$
-  $I_E = \dfrac{V_E}{R_E} \approx I_C$
-  $V_{CE} = V_{CC} - V_E$ （コレクタ側に抵抗が無いので R_C 項が入らない）

エミッタ接地と同じ  $V_{CE}=V_{CC}-I_C R_C$  を当てはめると誤る。📝FET ソース接地増幅の動作点（ソース自己バイアス）— R05下問18・R03問13・H24問18・H21問13

FET は電圧制御。ゲート電流 ≈ 0 なので V_G ≈ 0（ゲート抵抗 R_G があっても電圧降下ゼロ）。
-  $V_{GS} = V_G - V_S = 0 - I_D R_S = -\,I_D R_S$
- 特性式（I_D–V_GS）に代入して I_D を決定
- 電圧利得  $A_v = -\,g_m R_D$

BJT の V_BE≈0.6V のような固定値は無い（V_GS は動作点で変わる）。R_G を BJT のベース抵抗のように分圧計算しないこと。

## 1原理（なぜ起きるか）🔑直感アナロジー🚰BJT（バイポーラ）は「電流で電流を制御」する水道バルブ。ベース電流 I_B という小さい力でバルブを開閉すると、コレクタ電流 I_C = h_FE × I_B という大流量が流れる。
FET（電界効果型）は「電圧で電流を制御」するゲート弁。ゲートに電圧を加えるだけでチャネルが形成され、ドレイン電流が流れる。入力電流はほぼゼロ。

トランジスタの増幅動作は「アクティブ領域」で起こる。飽和領域ではスイッチON、遮断領域ではスイッチOFFとして扱う。直流動作点（Q点）は増幅動作が安定して続く動作点であり、バイアス回路で設定する。

> 図：BJT は小さなベース電流 I_B で大きなコレクタ電流 I_C を制御（電流制御）。FET はゲート電圧 V_GS で制御し、ゲート電流はほぼ 0（電圧制御）。「入力が電流か電圧か」が両者の最大の分岐点。

## 2なぜトランジスタで詰まりやすいか（メタ認知）

トランジスタは「用語は知っているのに正誤判定を外す」典型。原因は計算ミスではなく認知上の壁。症状と誤モデルを1表で対応付け、3節以降を読む前に「自分が今どこで詰まっているか」を意識する。

| 症状（つまずきポイント） | 誤モデル（脳内バグ） | 正モデル（突破の鍵） |
| --- | --- | --- |
| ① BJT と FET の制御方式を逆にする | 「トランジスタは全部電流で動く」 | BJT=電流制御・FET=電圧制御（→ 公式タブ 3節） |
| ② I_E = I_C と近似して I_B を無視 | 「h_FE≫1 だから I_B=0」 | I_E=I_B+I_C は全領域で省略不可。近似は最後の仕上げのみ（→ 公式タブ 1節） |
| ③ 電圧利得 = h_FE と思う | 「h_FE が大きい＝電圧利得が大きい」 | 電流利得=h_FE・電圧利得=R_C/h_ie は別の式（→ 5節引っかけ①） |
| ④ V_BE を引き忘れる | 「V_E = V_B でよい」 | V_E=V_B−0.6（Si接合電圧を必ず引く）（→ 公式タブ 2節） |
| ⑤ 接地方式の位相を一律に扱う | 「どの接地でも位相は変わらない」 | エミッタ接地のみ反転（180°）・ベース/コレクタ接地は非反転（→ 公式タブ 4節） |
| ⑥ FET を「電流が流れない素子」と極端解釈 | 「FET=絶縁=電流ゼロ」 | ゲート電流≈0 だがドレイン電流は流れる（→ 1節 原理） |

## 3直流動作点と電流帰還バイアスの「なぜ」💡🔑 1行で覚える本質

電流帰還バイアス＝「I_C が増えると V_E が上がり、ベースを抑える」自動ブレーキ（負帰還）。だから温度変化に強い。解き方・判断手順 （電流帰還バイアスの動作点）
- 分圧でベース電圧を固定: V_B = V_CC·R_2/(R_1+R_2)（h_FE に依存しない）
- V_E を求める: V_E = V_B − V_BE（Si は 0.6V を引く）
- I_C を求める: I_C ≈ I_E = V_E / R_E
- 温度で I_C が増えると V_E=I_E·R_E が上がる → V_BE=V_B−V_E が減る → I_B が減る → I_C が戻る（負帰還）
- 結論: 動作点が h_FE のばらつき・温度にほぼ依存せず安定する

> 図：分圧抵抗 R1・R2 でベース電圧 V_B を固定し、エミッタ抵抗 R_E が負帰還を生む。I_C が増えると V_E が上がって V_BE が減り、I_B を自動的に抑える。だから温度や h_FE のばらつきに動作点が左右されにくい。

## 4実務でどう活きる💡プラント電気・計装での使われどころトランジスタはセンサ信号の増幅・インターフェース回路・ドライバ回路として計装システム全体に組み込まれている。制御弁・ソレノイドバルブの駆動回路や、4〜20mA伝送の送信器内部でも核心部品として動作する。

| 現場シーン | 効いている物理 | 技術者の判断 |
| --- | --- | --- |
| 熱電対の微小電圧を制御器に伝送する増幅回路 | BJTのアクティブ領域動作（I_C=h_FE·I_B）で mV を mA に変換 | 動作点が温度変動でずれないよう電流帰還バイアスを採用 |
| ソレノイドバルブを PLC 出力で駆動するドライバ回路 | トランジスタを飽和領域でスイッチとして ON/OFF | コレクタ損失 P_C が最大定格を超えないようベース抵抗と放熱設計を確認 |
| フィールド機器の保護回路 | MOSFETの高入力インピーダンスで信号源への負荷を最小化 | V_GS がしきい値を常に確保できる電源電圧条件を確認して交換品を選定 |

## 5引っかけポイント✅正解の原則電圧利得は h_FE だけでなく、負荷抵抗 R_C と内部インピーダンス h_ie の比で決まる。⚠よくある誤解h_FE が大きければ電圧利得も自動的に大きいと思いがち。🔍判別ステップ
- 「電圧利得」か「電流利得」かを問題文で確認する
- 電流利得 → h_FE の値を直接使う
- 電圧利得 → h_ie（入力インピーダンス）と R_C の比で計算する（h_FE とは別の式）
- 混同しているなら次元を確認：利得に単位はなく、電流/電流 または 電圧/電圧出典: H28 問13 出題✅正解の原則I_E = I_B + I_C は全動作領域で常に成立する。I_E を I_C と完全に等しいと近似してはいけない。⚠よくある誤解h_FE ≫ 1 だから I_E ≈ I_C として I_B を無視しがち。🔍判別ステップ
- 問題文に「I_E を求めよ」とあれば必ず I_B + I_C を計算する
- 「I_C ≈ I_E」は許容近似だが「I_E = I_C（完全等号）」は誤り
- h_FE の値に関わらず、3端子の電流収支式は省略できない出典: （頻出引っかけ）✅正解の原則MOSFETのゲートに流れる電流は理論上ゼロ。バイアスは電圧だけで決まる。⚠よくある誤解BJT と同じようにゲート電流でバイアスを設計しようとする。🔍判別ステップ
- 使用素子が BJT か FET かを問題文で特定する
- FET なら入力電流 = 0 を前提に回路を解く
- MOSFETの動作条件は V_GS > V_th（しきい値電圧）の電圧条件で確認する出典: R05下 問11・H23 問11 出題

> 図：エミッタ接地だけ出力が入力と逆位相（180°反転）。ベース接地・コレクタ接地（エミッタフォロワ）は入力と同位相。「どの接地でも位相は同じ」と決めつけると失点する。✅正解の原則エミッタ接地の出力信号は入力と逆位相（180°）になる。ベース接地・コレクタ接地は非反転。⚠よくある誤解どの接地方式でも位相は変わらないと思いがち。🔍判別ステップ
- 「エミッタ接地」という接地方式名を問題文で確認する
- エミッタ接地 → 位相反転あり（入力が正のとき出力は負）
- ベース接地・コレクタ接地（エミッタフォロワ）→ 位相反転なし出典: H20 問13 出題

## 6拡張問い（思考を広げる）📍🧐 なぜ BJT は熱暴走し、FET は熱安定なのか？（diffuse mode 起動）

BJT は温度が上がると I_C が増える → 発熱が増える → さらに I_C が増える…という正帰還（熱暴走）の素質を持つ。だから電流帰還バイアス（負帰還）で抑える必要がある。一方 FET は温度上昇でキャリア移動度が下がりドレイン電流が減るため、自己安定（負帰還）が働く。もう一段深く（だから使い分けが生まれる）
- パワーMOSFET を並列接続できるのは、電流が偏った素子が熱で電流を減らし自動均等化するから（BJTは並列で一方に集中→破壊）
- BJT は同じ理由で「2次降伏」という破壊モードを持つ
- 「正帰還＝暴走／負帰還＝安定」は過渡現象・自動制御・発振回路にも共通する基礎概念← 前5.1 半導体次 →5.3 オペアンプ

## 2例題📊 このセクションの理解度:◎○△×📝例題1：電流増幅率と各電流の計算h_FE = 150 の NPN トランジスタにベース電流 I_B = 0.03 mA が流れているとき、コレクタ電流 I_C とエミッタ電流 I_E を求めよ。解答・解説 $I_C = h_{FE} \times I_B = 150 \times 0.03 = 4.5 \text{ [mA]}$  $I_E = I_B + I_C = 0.03 + 4.5 = 4.53 \text{ [mA]}$

答え：I_C = 4.5 mA、I_E = 4.53 mA（I_E = I_B + I_C は必ずキルヒホッフで導く。I_B をゼロにしない）📝例題2：電流帰還バイアス回路の動作点（R06上 問11 類題）V_CC = 12V、R_1 = 60 kΩ、R_2 = 20 kΩ、R_E = 1 kΩ の電流帰還バイアス回路で、コレクタ電流 I_C（≈ I_E）を求めよ。ただし V_BE = 0.6V、h_FE ≫ 1 とする。解答・解説 $V_B = 12 \times \dfrac{20}{60+20} = 3 \text{ [V]}$  $V_E = V_B - V_{BE} = 3 - 0.6 = 2.4 \text{ [V]}$  $I_C \approx I_E = \dfrac{V_E}{R_E} = \dfrac{2.4}{1000} = 2.4 \text{ [mA]}$

答え：I_C ≈ 2.4 mA（分圧 → V_E=V_B−0.6 → I_C=V_E/R_E の3ステップ。「0.6を引き忘れる」ミスが最多）📝例題3：FETの特徴選択（R06下 問11 類題）FET（電界効果トランジスタ）の特徴として正しいものを選べ。
① 電流制御型デバイスである／② ゲート入力電流は BJT のベース電流と同程度／③ MOSFETのゲートは絶縁され入力インピーダンスが非常に高い／④ BJTより温度安定性が低い解答・解説

① ❌ FET は電圧制御型（電流制御は BJT）。② ❌ MOSFETのゲートは酸化膜絶縁で入力電流ほぼゼロ。③ ✅ 正しい（絶縁ゲートで入力Z理論上∞）。④ ❌ FET は温度上昇でドレイン電流が減り、BJTより温度安定性が高い。

答え：③📝例題4：エミッタ接地増幅回路の電圧利得（R02問18・H21問18 類題）h_FE = 100 のトランジスタを I_C = 1 mA で動作させ、コレクタ抵抗 R_C = 3 kΩ とする。入力インピーダンス h_ie を V_T = 26 mV から見積もり、電圧利得 A_v を求めよ。解答・解説 $I_B = \dfrac{I_C}{h_{FE}} = \dfrac{1\,\text{mA}}{100} = 0.01\,\text{mA}$  $h_{ie} \approx \dfrac{V_T}{I_B} = \dfrac{26\,\text{mV}}{0.01\,\text{mA}} = 2.6\,\text{k}\Omega$  $A_v \approx -\,h_{FE}\dfrac{R_C}{h_{ie}} = -100\times\dfrac{3000}{2600} \approx -115$

答え：A_v ≈ −115（マイナス＝入力と逆位相）。電流利得 h_FE=100 と電圧利得は別物。h_ie が与えられなければ 26mV/I_B で見積もる。📝例題5：エミッタフォロワ（コレクタ接地）の動作点（R07上問18・H30問16 類題）V_CC = 12V、R_1 = 40 kΩ、R_2 = 20 kΩ、R_E = 1 kΩ、V_BE = 0.6V のエミッタフォロワで、V_E・I_E・V_CE を求めよ。解答・解説 $V_B = 12\times\dfrac{20}{40+20} = 4 \text{ [V]}$  $V_E = V_B - V_{BE} = 4 - 0.6 = 3.4 \text{ [V]}$  $I_E = \dfrac{V_E}{R_E} = \dfrac{3.4}{1000} = 3.4 \text{ [mA]} \approx I_C$  $V_{CE} = V_{CC} - V_E = 12 - 3.4 = 8.6 \text{ [V]}$

答え：V_E=3.4V、I_E≈3.4mA、V_CE=8.6V。コレクタ側に抵抗が無いので V_CE=V_CC−V_E（R_C 項を入れると誤り）。📝例題6：FET ソース接地増幅（R05下問18・H21問13 類題）相互コンダクタンス g_m = 2 mS、ドレイン抵抗 R_D = 5 kΩ のソース接地増幅回路の電圧利得 A_v を求めよ。また I_D = 2 mA・R_S = 0.5 kΩ のときのソース自己バイアス V_GS を求めよ。解答・解説 $A_v = -\,g_m R_D = -\,2\times10^{-3}\times 5000 = -10$  $V_{GS} = -\,I_D R_S = -\,2\,\text{mA}\times 0.5\,\text{k}\Omega = -1 \text{ [V]}$

答え：A_v=−10倍、V_GS=−1V。ゲート電流≈0 なので V_G=0 → V_GS=−I_D R_S。BJT の V_BE のような固定値は無い。

## 3R06上問11 解法カード — BJT vs FET の正誤判定📊 このセクションの理解度:◎○△×令和6年上期問11

バイポーラトランジスタと電界効果トランジスタ（FET）に関する記述として、誤っているものを次の(1)〜(5)のうちから一つ選べ。
- (1) バイポーラトランジスタは、消費電力がFETより大きい。
- (2) バイポーラトランジスタは、静電気に対してFETより破壊されにくい。
- (3) バイポーラトランジスタの入力インピーダンスは、FETのそれよりも低い。
- (4) バイポーラトランジスタは電圧制御素子、FETは電流制御素子といわれる。 ← 誤り（これが正解）
- (5) バイポーラのコレクタ電流は電子と正孔の両方、FETのドレーン電流はどちらか一方が関与。「誤っているもの」を選ぶ問題。本ページの核心 BJT=電流制御 / FET=電圧制御 を逆にした選択肢が答え。解き方・判断手順 （正誤判定（BJT vs FET））
- 各記述を「BJT の性質」「FET の性質」に分けて読む
- (1) 消費電力 BJT>FET：正しい（FETは低消費電力）
- (2) 静電破壊：BJT は強い・FET（絶縁ゲート）は弱い → 正しい
- (3) 入力インピーダンス BJT<FET：正しい（FETは高入力Z）
- (4) 制御方式：BJT=電流制御・FET=電圧制御 が正。記述は逆 → 誤り＝答え
- (5) キャリア：BJT は電子+正孔（bipolar）・FET は一方（unipolar）→ 正しい📝 まず自分で解答 → 👁 正解・解説を見る過去問は常に即表示にする⚠本問のひっかけポイント（5項目）
- 「誤っているもの」を選ぶ問題で正しい記述を選ぶ（誤）⇒ 設問の「正/誤」を最初に丸で囲む
- BJT と FET の制御方式を逆に覚えている（誤）⇒ BJT=電流・FET=電圧（本問の核心）
- 静電破壊の強弱を逆にする（誤）⇒ 絶縁ゲートの FET が静電気に弱い
- bipolar / unipolar の意味を取り違える（誤）⇒ BJT は2種キャリア、FET は1種
- 入力インピーダンスの大小を逆にする（誤）⇒ FET が高入力インピーダンス
