- スイッチONの瞬間、C に蓄えた電荷 Q と L に流れる電流 iL は直前の値（ここではゼロ）を引き継ぐ（蓄えは一瞬では変われない）

スイッチ投入直後の回路は 過渡状態。電流・電圧が定常値に近づく速さを表すのが 時定数 τ：

 $\tau \;=\; RC\quad\text{(RC回路)}, \qquad \tau \;=\; \dfrac{L}{R}\quad\text{(RL回路)}\quad[\mathrm{s}]$

単位は 秒 s。等価的に 「定常値の 63.2%（= 1 − e−1）に到達するまでの時間」＝ τ。実用的には 3τ で 95%、5τ で 99.3% 定常到達と扱う（OFF時は逆に 36.8% へ減衰）。

因果でつなぐと τ 大 → 過渡応答が遅い → 定常到達まで時間がかかる。RC は C が大きいほど（電圧変化を強く嫌う）／R が大きいほど（電流を流しにくい）遅くなる。RL は L が大きいほど（電流変化を強く嫌う）／R が小さいほど遅くなる。τ は 素子の「変化を嫌う強さ」と R の関係で決まる時間スケールで、交流 ω の逆数的性質を持つ（ω 大＝速い振動 ↔ τ 大＝遅い緩和）。🧩 チャンク階層 — どこを覚えているか自己確認第1層τ（時定数）変化の速さ1個: τ=RC / τ=L/R第2層f(0)・f(∞)2点を押さえる: 初期値と最終値第3層f(t)=f(∞)+[f(0)−f(∞)]e−t/τ指数で2点を結ぶ一般式第1層 → 第2層 → 第3層 の順で自己診断。「一般式の使い方で詰まる」なら第3層、「初期値・最終値の設定で詰まる」なら第2層、「τ=RC か L/R で詰まる」なら第1層から復習。💡⚡ 5秒で思い出す

- 時定数 τ：RC回路は τ=RC・RL回路は τ=L/R（変化の速さ）

## 5RLC回路の素子を短絡したときの回路変化（R07上・R06上・R04下 問10）

| 短絡する素子 | 残る回路型 | 時定数 | 過渡現象の変化 |
| --- | --- | --- | --- |
| C を短絡 | RL回路（1次） | $\tau = L/R$ | RL過渡へ移行。C 両端電圧は 0 に向かう |
| L を短絡 | RC回路（1次） | $\tau = RC$ | RC過渡へ移行。L の電流は 0 に向かう |
| R を短絡 | LC回路（2次） | —（時定数なし） | 減衰なしの振動。抵抗が消え損失ゼロ → L と C の間でエネルギーが往復し電流が持続 |
⚠R04下問10（論説）の急所

R を短絡（バイパス）する問題では、「エネルギーを消費する場所（抵抗）がなくなるため電流が減衰しなくなる」「L と C の間でエネルギーが行き来する振動状態になる」という定性理解が問われる。R07上・R06上の計算問題では、短絡後の  $\tau = L/R$  または  $\tau = RC$  に切り替えて解き直すのがポイント（短絡前の τ を流用しない）。詳しい2次回路の減衰振動は RLC回路 を併読。

## 6RC充電のエネルギー — ちょうど半分則（R01問10）

直感に反する頻出の引っかけ。電源から供給される総エネルギーのちょうど半分が抵抗で熱になり、残り半分がコンデンサに蓄えられる。しかもこれは抵抗値 R の大きさによらず成立する。⚠供給エネルギーは 1:1 に分かれる（R に依らない）

初期電荷ゼロの RC 直列回路を電圧 E で充電完了させると： $W_R = W_C = \tfrac{1}{2}CE^2, \qquad W_{\text{total}} = CE^2$

コンデンサに蓄えられる  $W_C = \tfrac{1}{2}CE^2$  と、抵抗で消費される  $W_R = \tfrac{1}{2}CE^2$  が常に等量。「R を大きくすれば損失が増えそう」という直感は誤り。R を変えても両者の比は 1:1 のまま（R は充電の時定数 τ=RC を変えるだけで、総損失量は変えない）。

| 素子 | 蓄えられるエネルギー |
| --- | --- |
| コンデンサ C（電圧 V のとき） | $\dfrac{1}{2}CV^2$ |
| インダクタ L（電流 I のとき） | $\dfrac{1}{2}LI^2$ |

## 1原理（なぜ起きるか）🔑直感アナロジー💧貯水タンクに水を注ぐイメージ：最初は勢いよく増え、満杯に近づくほどゆっくりになる。コンデンサへの充電やコイルへの電流増加も同じ指数カーブを描く。時定数 τ（タウ）は「変化の速さ」の物差しで、τ 秒後に「残り変化量の約37%が残っている」（≒ 63%変化した状態）。

解法の鉄則は初期値と最終値を先に求めること。あとは一般式  $f(t) = f(\infty) + [f(0) - f(\infty)]\,e^{-t/\tau}$  に当てはめるだけ。τ 後 = 63%変化、5τ 後 ≒ 99%（実用上の定常到達目安）。

## 6引っかけポイント⚠勘違い①：t=0直後にコンデンサを「開放」として扱う初期電荷ゼロのコンデンサは t=0⁺ で短絡に等しい（電圧=0を維持）。電流は最大値で流れ始め、徐々に減少する。コンデンサは「電荷を蓄えるから電流が流れない」ではなく、「電圧が急変できない」素子。⚠勘違い②：1τ後に定常値に達したと思い込む1τ 後は約63%変化した状態。100%に近づくのは理論上無限大時間かかる。実用上は 5τ 後（≒99%）を定常到達の目安とする。「τ で定常達成」は誤り。⚠勘違い③：RL回路で t=0直後のインダクタを「短絡」として扱う初期電流ゼロのインダクタは t=0⁺ で開放に等しい（電流は急変できない）。電流はゼロから徐々に増加する。⚠勘違い④：RC回路とRL回路の時定数式を取り違える次元チェックが有効。 $\tau = L/R$  は [H]/[Ω]=[s]、 $\tau = CR$  は [F]·[Ω]=[s] と確認する。RL回路の微分方程式を解くと指数係数は R/L → 時定数は L/R。📝複雑な回路の時定数はテブナン抵抗で求める（R02問10）C または L を取り外し、全電圧源を短絡・全電流源を開放して端子間の合成抵抗 R_th を求める。時定数は  $\tau = R_{th}C$ （RC回路）または  $\tau = L/R_{th}$ （RL回路）。

## 7拡張問い（思考を広げる）📍🧐 なぜ指数関数になるのか？（diffuse mode 起動）

RC充電では  $\dfrac{dv_C}{dt} = \dfrac{E - v_C}{RC}$ 。つまり変化の速さが「残りの差（E−vC）」に比例する。満杯に近いほど差が小さく、変化がゆっくりになる。この「自分の残差に比例して減る」構造の解が指数関数 e−t/τ。もう一段深く（同じ構造の自然現象）
- 放射性崩壊：残っている原子数に比例して崩壊 → 同じ指数則（半減期 ↔ 時定数）
- ニュートンの冷却：温度差に比例して冷える → 指数で常温へ
- いずれも1階線形微分方程式。過渡現象を理解すれば物理・化学の指数現象が一気に繋がる
- 逆に L と C が両方あると2階になり、指数ではなく減衰振動（RLC回路）に変わる← 前3.1 インダクタンス次 →3.3 三相交流

## 1過渡現象問題の解法フロー（汎用5ステップ）📊 このセクションの理解度:◎○△×

過渡現象の過去問はほぼこのフローで解ける。個別の解法カードに進む前に、まずこの汎用フローでパターンを抽出すること。解き方・判断手順 （過渡現象（汎用））
- 回路判別: RC か RL か・スイッチ前後でどの状態が変わるか
- 時定数 τ: τ=RC または L/R（複数Rの複雑回路はテブナン抵抗 R_th で τ=R_th·C / L/R_th）
- 初期値 f(0): スイッチ直前から連続量（v_C・i_L）を引き継ぐ。t=0⁺等価回路（初期ゼロなら C短絡・L開放／初期値ありは電圧源・電流源）
- 最終値 f(∞): t=∞等価回路（C開放・L短絡）で直流計算
- 一般式に代入: f(t)=f(∞)+[f(0)−f(∞)]e^(−t/τ)・τ後63.2%で検算・単位[s]を確認

## 2例題📊 このセクションの理解度:◎○△×

問1: R=1kΩ、C=100μF のRC直列回路にDC10Vを印加した。時定数 τ と、τ 後のコンデンサ電圧を求めよ。解答 $\tau = RC = 1000 \times 100 \times 10^{-6} = 0.1\text{ s}$  $v_C(\tau) = 10(1 - e^{-1}) \approx 10 \times 0.632 = 6.32\text{ V}$

ポイント：τ 後には最終値の約63.2%まで充電される。

問2: スイッチ切り替え前の定常状態でコンデンサに8Vが充電されている。切り替え後の回路では最終値が0V、時定数が0.05sである。切り替え後の v_C(t) を求めよ。解答

初期値  $f(0) = 8\text{ V}$ 、最終値  $f(\infty) = 0\text{ V}$ 、 $\tau = 0.05\text{ s}$  $v_C(t) = 0 + [8 - 0]\,e^{-t/0.05} = 8\,e^{-20t}\text{ V}$

ポイント：切り替え前の電圧が切り替え後の初期値になる（コンデンサ電圧は急変不可）。

## 3R06下問10 解法カード — 過渡/定常の電流比📊 このセクションの理解度:◎○△×令和6年下期問10

電源 E [V] に 6Ω が直列に接続され、その先に「4Ω とコンデンサの直列枝」と「12Ω」が並列に接続されている。点A（6Ω を流れる位置）の電流を、スイッチSを閉じた瞬間（t=0）に I₀、十分時間が経った定常状態で I とする。電流比 I₀/I に最も近いものを選べ。コンデンサの初期電荷は零とする。
- ① 0.5
- ② 1.0
- ③ 1.5
- ④ 2.0  ← 正解
- ⑤ 2.5t=0⁺ は C短絡・t=∞ は C開放。この置き換えで合成抵抗が変わる典型問題。解き方・判断手順 （過渡/定常の電流比）
- t=0⁺ 等価回路: コンデンサ短絡 → 4Ω と 12Ω が並列 = 4×12/(4+12) = 3Ω
- I₀ 算出: 合成 R₀ = 6 + 3 = 9Ω → I₀ = E/9
- t=∞ 等価回路: コンデンサ開放 → 4Ω 枝に電流が流れず → 12Ω のみ有効
- I 算出: 合成 R = 6 + 12 = 18Ω → I = E/18
- 比: I₀/I = (E/9)/(E/18) = 18/9 = 2.0（E は約分で消え、値は不要）📝 まず自分で解答 → 👁 正解・解説を見る過去問は常に即表示にする⚠本問のひっかけポイント（5項目）
- t=0 でコンデンサを開放扱い（誤）⇒ 初期電荷ゼロの C は t=0 で短絡。電流は最大
- t=∞ でコンデンサを短絡扱い（誤）⇒ 定常では C 開放。4Ω 枝は電流ゼロで 12Ω のみ効く
- 4Ω と 12Ω を直列にする（誤）⇒ t=0 では両者は並列（3Ω）
- 電流比の向きを I/I₀ と逆に取る（誤）⇒ 問われるのは I₀/I（過渡÷定常）
- 電源電圧 E の値を探して止まる（誤）⇒ E は比で約分。値が無くても解ける

## 43条件即答チェック（限界的練習ゾーン）— RC充電の時間発展📊 このセクションの理解度:◎○△×

「解ける／解けない」の境界を強制発火させる練習。同一RC回路で経過時間を3条件変えて即答し、典型誤答3択でセルフ判定。3条件すべて正答できれば B 問題対応 OKの目安。📍共通設定

E = 10 V・R = 2 kΩ・C = 50 μF の RC直列回路に DC を印加（初期電荷ゼロ）。τ = RC = 0.1 s・E/R = 5 mA。充電電圧 vC=E(1−e−t/τ)・電流 i=(E/R)e−t/τ。

即答してから下の解答を開くこと（暗算 30 秒目安）。

| 条件 | vC [V] | 充電率 | 電流 i [mA] |
| --- | --- | --- | --- |
| ① t = τ（0.1 s） | 6.32 | 63.2 % | 1.84 |
| ② t = 2τ（0.2 s） | 8.65 | 86.5 % | 0.68 |
| ③ t = 3τ（0.3 s） | 9.50 | 95.0 % | 0.25 |
解答の導出（3条件まとめて）

① t=τ: 1−e−1=0.632 → vC=10×0.632=6.32V・i=5×0.368=1.84mA

② t=2τ: 1−e−2=0.865 → vC=10×0.865=8.65V・i=5×0.135=0.68mA

③ t=3τ: 1−e−3=0.950 → vC=10×0.950=9.50V・i=5×0.0498=0.25mA

※ 電圧は上がるほど電源との差が縮むので、電流は逆に減衰する。⚠典型誤答3択（自己採点用）
- ① で「100%充電」と答える（誤）⇒ τ 後は 63.2%。τ は時定数で完了時間ではない
- 電流 i が時間とともに増えると答える（誤）⇒ 充電電流は減衰（e−t/τ）。電圧上昇で電位差が減り電流も減る
- vC(2τ)=2×vC(τ) と答える（誤）⇒ 指数なので比例しない。6.32×2=12.64V は E=10V を超え物理的に不可能💡セルフ判定

3条件すべて即答 ⇒ 充電・放電の時間発展 B 問題対応 OK。1問でも誤答 ⇒ 解説タブ 3節グラフ・4節（初期値・最終値）を再読。
