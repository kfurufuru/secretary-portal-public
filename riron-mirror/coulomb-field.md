## 3電界と電位の関係 — 「比例」ではなく「勾配」

電界は電位の空間的な変化率（勾配）であり、 $E = -\dfrac{dV}{dr}$ （1次元）。電位の絶対値ではなく「どれだけ急激に変化しているか」が電界の強さを決める。
- 電位が高くても変化が緩やかなら電界は弱い。電位が低くても急勾配なら電界は強い
- 等電位面上では電位変化がゼロ → その方向に電界は存在しない → 電界（力線）は常に等電位面と垂直
- マイナス符号の意味：電界は「電位が下がる向き」を向く（山を下る向き）⚠頻出の罠文章（電位と電界の関係）「電位 V [V] が大きい点ほど電界 E [V/m] も強い」← 誤り。E は V の勾配であって比例ではない。

> 図：点電荷が作る電位 V（∝1/r）と電界 E（∝1/r²）。E は V より速く減衰し、E は V の傾き（E=−dV/dr）。だから「V が大きい点＝E が強い点」ではなく、V の傾きが急な場所ほど E が強い。

## 4電気力線の読み方（本数から電荷を逆算する）

公式タブ 3節の性質表のうち、計算問題に直結するのは「本数 N = Q/ε₀」。
- なぜ交差しないか：電界の向きは各点でただ1つ。交差したら「その点の力の向きが2つある」ことになり矛盾
- 本数2倍 → 電界2倍：力線密度 ∝ 電界の強さ E（H23・R05上で出題された論点）
- H19問3 の典型パターン：閉曲面から出る力線が N 本なら内部電荷は Q = Nε₀。出る力線と入る力線が混在する場合は正味の本数（出る − 入る）で計算

## 5複数点電荷 — 合力・釣り合い位置・電位ゼロ点

合力（ベクトル和）の計算手順：
- 各電荷からのクーロン力 F の大きさを求める
- 各力の向きを座標軸で分解する（x成分・y成分）
- 成分ごとに代数和をとり、合ベクトルの大きさと向きを求める（正三角形配置（H20問1・H22問17）は対称性で打ち消しを先に確認）

釣り合い位置（R07上・R04下・H20 問17で繰り返し出題）：同符号の2電荷 Q₁・Q₂（距離 d）の間に電荷 q を置いて釣り合わせる。Q₁ から距離 x の点で：

 $\dfrac{Q_1 q}{4\pi\varepsilon_0 x^2} = \dfrac{Q_2 q}{4\pi\varepsilon_0 (d-x)^2} \;\Rightarrow\; \dfrac{\sqrt{Q_1}}{x} = \dfrac{\sqrt{Q_2}}{d-x} \;\Rightarrow\; x = \dfrac{\sqrt{Q_1}}{\sqrt{Q_1}+\sqrt{Q_2}}\, d$ ⚠釣り合い位置の3つの引っかかり
- Q₁ = Q₂ なら中点（x = d/2）
- 同符号なら釣り合い位置は必ず2電荷の間。外側に求まったら計算ミスを疑う
- Q₁ ≠ Q₂ のとき釣り合い位置は小さい電荷に近い側（x は √Q₁ に比例＝大きい電荷からは遠ざかる）

3電荷の同時平衡＝電気量比（H5-B問21・旧制度の良問）：等間隔 ℓ で一直線上に固定した3電荷 Q₁・Q₂・Q₃ が、3つとも力ゼロになる電気量比を求める「逆問題」。上の釣り合い位置が位置を求めるのに対し、こちらは位置を固定して電気量比を求める。核心は逆二乗則（距離2倍＝力1/4）。
- 中央 Q₂ の釣り合い → Q₁ = Q₃：Q₂ から見ると Q₁・Q₃ は左右に等距離 ℓ。2力が打ち消すには大きさが等しく同符号 ⇒ Q₁ = Q₃。
- 端 Q₁ の釣り合い → Q₂ = −Q₁/4：Q₁ には Q₂（距離 ℓ）と Q₃（距離 2ℓ）が働く。逆二乗で Q₃ の寄与は 1/(2)² = 1/4。釣り合いから大きさ |Q₂| = Q₁/4、向きを打ち消すため Q₂ は外側と逆符号。

 $\dfrac{Q_1 Q_2}{4\pi\varepsilon_0 \ell^2} = \dfrac{Q_1 Q_3}{4\pi\varepsilon_0 (2\ell)^2} \;\Rightarrow\; |Q_2| = \dfrac{Q_3}{4} = \dfrac{Q_1}{4} \;\Rightarrow\; Q_1 : Q_2 : Q_3 = 4 : -1 : 4$

> 図：3電荷を等間隔 ℓ で固定。中央 Q₂ は左右等距離の引力で自明に相殺。端の Q₁ は、近い中央の引力（距離 ℓ）と遠い Q₃ の斥力（距離 2ℓ）が釣り合う。距離が2倍になると力は 1/2² = 1/4 に弱まるので、中央は外側の 1/4・逆符号 でちょうど平衡 → Q₁:Q₂:Q₃ = 4 : −1 : 4。

> 図：等間隔 ℓ に +4, −1, +4 を置くと3電荷すべてが静止。端の Q₁ では、遠い Q₃ の斥力（距離2ℓで 1/4）と近い Q₂ の引力（距離ℓ）がともに 4k/ℓ² で打ち消す。これが 4 : −1 : 4 の正体。⚠3電荷平衡の引っかかり
- 最頻トラップ＝1 : −4 : 1：1/4 を中央でなく外側に乗せる取り違え。正しくは外側が大きく中央が小さい（遠い相手の弱い反発を、近い中央で受け止めればよいから中央は小さくて足りる）。
- 逆二乗を忘れる（1/r 処理）と 2 : −1 : 2（中央＝外側の 1/2）に誤る。正しくは 1/4 で 4 : −1 : 4。
- 外側2電荷を同符号にしないと中央が打ち消せない。全部同符号（1:1:1）や中央ゼロ（1:0:−1）では平衡しない。

※ この平衡は不安定平衡（少しずれると崩れる）。ただし電気量比を問う本問では不問。

電位がゼロとなる点（H22問1）：異符号の2電荷（+Q₁ と −Q₂）では、電位はスカラーなので代数和 = 0 で解く：

 $V = \dfrac{Q_1}{4\pi\varepsilon_0 r_1} - \dfrac{Q_2}{4\pi\varepsilon_0 r_2} = 0 \;\Rightarrow\; \dfrac{Q_1}{r_1} = \dfrac{Q_2}{r_2}$
- 異符号 → V=0 の点は一般に 2箇所（2電荷の内側に1点・外側に1点）
- 同符号 → V=0 の点は存在しない（両方正なら V は常に正）

## 6静電遮蔽（静電シールド）

原理：静電平衡状態の導体の内部電界はゼロ（なぜゼロか＝「電荷ゼロ」だけでなく「対称性」との合わせ技。8節 拡張問いの「ガウスの法則」参照）。導体で囲まれた空洞内部も、空洞内に電荷がなければ電界ゼロ → 外部の影響が中に及ばない＝静電遮蔽。

メカニズム：
- 外部に電界をかける → 導体外表面に電荷が誘導される
- 誘導電荷が外部電界を完全に打ち消す方向の電界を作る
- 結果として空洞内部の電界 = 0

空洞内に電荷 +Q を置いた場合：空洞内側の導体表面に −Q が誘導され、外側表面に +Q が現れる → 内部の帯電体の影響は外部には漏れる。

接地すると：外側表面の +Q が大地へ逃げ、外部への電界が完全にゼロ（完全遮蔽）。内側の −Q は内部の +Q と引き合っているので動かない。

| 状況 | 空洞内の電界 | 外部への電界 |
| --- | --- | --- |
| 導体に囲まれた空洞（空洞内電荷なし） | ゼロ | 外部電界による誘導あり |
| 空洞内に電荷 +Q（接地なし） | +Q の影響あり | 外側表面の +Q による電界あり |
| 空洞内に電荷 +Q（導体を接地） | +Q の影響あり | ゼロ（外部遮蔽完成） |
| 導体球の内部（電荷なし） | ゼロ | 外部電界を受けて表面に誘導電荷 |
⚠静電遮蔽のよくある勘違い
- ❌「導体シールドは電流を流すから電界を打ち消せる」→ ✅ 静電遮蔽は電流ではなく電荷の再分布（静電誘導）。静電平衡状態で成立
- ❌「接地しなくても外部への電界は遮蔽できる」→ ✅ 空洞内に電荷がある場合、接地しないと外表面に電荷が現れ外部に電界が漏れる

## 7引っかけ・誤答パターン⚠電界の向きは電子の動く向きではない電界の向きは「正電荷が受ける力の向き」＝ +極（高電位）から −極（低電位）へ。電子の移動方向は逆（−から+へ移動）。⚠誘電体中はε₀→ε₀ε_r に置換（R03問2 論説）絶縁性液体（比誘電率 ε_r）で満たした容器中の2小球間の静電力は F = Q₁Q₂/(4πε₀ε_r r²)。ε_r＞1 なので真空中より弱くなる（1/ε_r 倍）。引力・斥力の向きは変わらず大きさだけ変わる。⚠電位公式の r は「電荷からの距離」V = Q/(4πε₀r) の r は点電荷そのものから着目点までの距離。問題文の「電極間距離」や「導線の長さ」をそのまま入れない。💡仕事が問われたら「まず電位を求める」電位はスカラーなので向きを気にせず足し算できる。W=QΔV が最短手順（R02問1 型）。電界の積分を直接計算しようとすると時間を消費する。

## 8拡張問い（深掘り）ガウスの法則 — 点電荷公式の「親」と「内部 E=0」の本当の理由

閉曲面を貫く電束（電界×面積）= 内部電荷 / ε₀： $\oint \vec{E} \cdot d\vec{S} = \dfrac{Q_{enc}}{\varepsilon_0}$ 。 点電荷の電界 E = Q/(4πε₀r²) は、半径 r の球面（表面積 4πr²）にガウスの法則を適用すると直接導出できる。電験3種では公式として使えば足りるが、以下2点が本質。

① なぜ右辺が Q/ε₀ なのか（r² が消える）：球面を貫く電束は  $E \times 4\pi r^2 = \dfrac{Q}{4\pi\varepsilon_0 r^2}\times 4\pi r^2 = \dfrac{Q}{\varepsilon_0}$ 。電界が 1/r² で弱まる分、球面積が r² で広がるので、掛けると r² が相殺する。だから「囲む面の大きさ・形によらず、貫く総数は中の電荷だけで決まる」。これが力線本数 N = Q/ε₀ の正体。💡② 本質：Q内部=0 は「電界ゼロ」ではなく「正味フラックスがゼロ」

ガウスの法則が言うのは 面を出入りする力線の差し引き（正味フラックス）がゼロ であって、各点の E がゼロという意味ではない。何もない空間にガウス面を置くと、力線が片側から入り反対へ抜ける → 差し引きゼロだが E ≠ 0。「電荷がない＝電界ゼロ」と短絡すると、この例で矛盾する。

では導体球の内部でなぜ E=0 か → 対称性が効く。 球対称だとガウス面上で E はどこでも同じ大きさ・放射方向に縛られる。すると正味フラックス = E×4πr² となり、内部電荷ゼロから E×4πr²=0 → E=0。離れた空間の点には E を一様に縛る対称性がないので、Q内部=0 でも E≠0 のまま。E=0 の理由は「電荷ゼロ」＋「対称性」の合わせ技（→ 6節 静電遮蔽・導体内部）。実務との接点 — 絶縁破壊は電界集中点で起きる

高電圧機器（変圧器・ケーブル）の絶縁設計は電界分布の設計。絶縁破壊は電界が集中する点（角・細い曲面）で発生するため、シールド電極やテーパー形状で電界を均一化する。静電遮蔽は計測器・精密機器の金属筐体（外部静電界から内部回路を守る）の基礎。磁気シールドは別概念（強磁性体を使う）。

## 1静電気問題の解法フロー（汎用5ステップ）📊 このセクションの理解度:◎○△×

静電気の計算問題は「何を問われているか」の判別が9割。ベクトル（F・E）かスカラー（V・W）かで手順が分かれる。解き方・判断手順 （静電気（クーロン力・電界・電位））
- 問われている量を判別: 力 F／電界 E（ベクトル）か、電位 V／仕事 W（スカラー）か
- 媒質を確認: 真空・空気なら ε₀（k=9×10⁹）・「比誘電率」とあれば ε₀→ε₀ε_r
- 公式選定: F=kQ₁Q₂/r²・E=kQ/r²・V=kQ/r（V だけ 1/r）・W=QΔV
- 単位変換して代入: μC=10⁻⁶ C・cm=10⁻² m を先に直す。r は中心間距離
- 検算: 向き（同符号=斥力/異符号=引力）と桁（10のべき）を選択肢と照合

静電気の「向き」も同じ判定フローで出せる。「何の向きか」を見分ければ、暗記に頼らず向きが決まる。

- ⑤ 0.124 ← 正解同じ大きさの導体球同士の接触 = 合計電荷を等分配。出典: 電験王 (denken-ou.com/rironr7-1-17)解き方・判断手順 （接触による電荷分配 + 釣り合い位置）

- 解の公式: x = (−0.6+√0.72)/2 ≈ 0.124 m。A′(小さい電荷)側に寄ることを確認📝 まず自分で解答 → 👁 正解・解説を見る過去問は常に即表示にする⚠本問のひっかけポイント（5項目）
- 接触の見落とし：元の電荷 4 と 6 のまま釣り合い計算をすると別の値になる。「接触」とあったら必ず等分配後の電荷で計算
- 等分配は「同じ大きさの導体球」が条件：大きさが違えば等分にならない（容量比で分配）
- 2回目の接触は「2+6=8 の等分」：Cが1回目で得た 2×10⁻⁸ を持ったままBに触れる。6 の等分（3ずつ）ではない

## 43条件即答チェック（限界的練習ゾーン）📊 このセクションの理解度:◎○△×

真空中に点電荷 Q = 4μC。距離 r = 0.3 m の点 P を考える。k = 9×10⁹ N·m²/C²。

即答してから下の解答を開くこと（暗算 30 秒目安）。

| 条件 | 求める量 | 使う式 |
| --- | --- | --- |
| ① 電界 | P 点の電界の強さ E [V/m] | E = kQ/r² |
| ② 電位 | P 点の電位 V [V] | V = kQ/r |
| ③ 力 | P 点に q = 1μC を置いたとき受ける力 F [N] | F = qE |
解答の導出（3条件まとめて）

① 電界: E = 9×10⁹ × 4×10⁻⁶ / 0.3² = 3.6×10⁴ / 0.09 = 4×10⁵ V/m

② 電位: V = 9×10⁹ × 4×10⁻⁶ / 0.3 = 3.6×10⁴ / 0.3 = 1.2×10⁵ V

③ 力: F = qE = 1×10⁻⁶ × 4×10⁵ = 0.4 N（検算: F = kQq/r² = 9×10⁹×4×10⁻¹²/0.09 = 0.4 N で一致）⚠典型誤答3択（自己採点用）
- ② で「V = 4×10⁵ V」と答えた（誤）⇒ それは E の値。V = kQ/r は r の1乗で割る（E だけが r²）
- 桁が10⁶ ずれた（誤）⇒ μC の変換忘れ。代入前に 4μC = 4×10⁻⁶ C へ直す習慣をつける
- ③ で「F = 1.6 N」と答えた（誤）⇒ F = qE の q に源の 4μC を入れている。力を受けるのは置かれた電荷 q = 1μC💡セルフ判定
