使う記号: C（静電容量・F）／Q（電荷・C）／V（電圧・V）／W（蓄積エネルギー・J）／S（極板面積・m²）／d（極板間距離・m）／ε₀（真空誘電率 8.85×10⁻¹² F/m）／ε_r（比誘電率・無次元・≥1）／E（電界・V/m）／σ（面電荷密度・C/m²）📝📘 静電容量 C とは（定義）

コンデンサに電圧 V を加えると電荷 Q が蓄えられる。両者の比が 静電容量 C：

 $C \;=\; \dfrac{Q}{V}\quad[\mathrm{F}]$

単位は ファラド F ＝ C/V ＝ A·s/V。等価な見方として、 $i = C\,dV/dt$  から 「毎秒 1 V の電圧変化で 1 A の電流を流すコンデンサが 1 F」とも定義できる。

因果でつなぐと C が大きい → 同じ電圧でより多くの電荷を蓄える → 電圧変化に対して大きな充放電電流が流れる → 電圧変化に強く逆らう。C は「コンデンサの定数」ではなく 電圧変化への抵抗の強さそのもの。抵抗 R（電流そのものを妨げ熱消費）／コイル L（電流の変化を妨げ磁界蓄積）との R／L／C 双対構造のうち、C は 電界にエネルギーを蓄える側を担う。🧩 チャンク階層 — どこを覚えているか自己確認第1層Q・V・C定義: C = Q/V · 1コンデンサの基本3量第2層S・d・ε構造: C = ε₀ε_r·S/d · 平行平板の容量決定要因第3層合成・蓄積応用: 直列1/C・並列C・W=CV²/2・誘電体ε_r倍第1層 → 第2層 → 第3層 の順で迷子を自己診断。直/並列で迷ったら第3層・公式で迷ったら第2層・定義で迷ったら第1層。💡⚡ 5秒で思い出す

コンデンサ ＝ 水槽。容量 C = 水槽サイズ・電圧 V = 水位・電荷 Q = 水量。エネルギーは電界に蓄える。
- 定義：C = Q/V（1V で何クーロン溜まるか）
- 構造：C = ε₀ε_r·S/d（面積大・間隔小・誘電率大で容量増）
- エネルギー：W = CV²/2 = Q²/(2C) = QV/2（全部同じ値）
- 合成：直列は逆数の和（小さくなる）／並列は足し算（大きくなる）— 抵抗と逆転
- 誘電体挿入：定電圧なら Q・W が ε_r 倍／定電荷なら V・W が 1/ε_r 倍

## 32種類の誘電体挿入（縦/横）

R04上問17 / R06下問1 出題。極板に「垂直」=直列等価、極板に「平行」=並列等価。

| 配置 | 等価回路 | 合成容量 | 等しい量 |
| --- | --- | --- | --- |
| 縦入れ（厚さ方向）誘電体を上下に積層 | 直列 | $\frac{1}{C} = \frac{d_1}{\varepsilon_0 \varepsilon_{r1} S} + \frac{d_2}{\varepsilon_0 \varepsilon_{r2} S}$ | 各層の電荷 Q（直列）電界 E は誘電率に反比例 |
| 横入れ（面積方向）誘電体を左右に分割 | 並列 | $C = \frac{\varepsilon_0 \varepsilon_{r1} S_1}{d} + \frac{\varepsilon_0 \varepsilon_{r2} S_2}{d}$ | 各領域の電圧 V（並列）電界 E も等しい（V/d） |

> 図：誘電体が上下に積み重なる＝直列（1/C の和）／左右に並ぶ＝並列（C の和）。極板（灰）に対する誘電体の入り方で見分ける。⚠見分け方

図で「誘電体が上下に積み重なる」→ 直列（1/C の足し算）。「誘電体が左右に並ぶ」→ 並列（C の足し算）。

## 4導体板（金属板）挿入

H26問1 / H18問17 出題。厚さ t の導体板を挿入すると実効的な極板間距離が縮まり容量が増加。

> 図：導体板内は電界 0。電界が生じるのは上下の空気層だけ → 実効的な極板間距離は d − t に縮み、容量が増える。

 $C' = \varepsilon_0 \frac{S}{d - t}$

n 枚の金属板（厚さ t）を等間隔で挿入する場合、(n+1) 個直列接続と等価。各コンデンサの間隔 d' = (d - nt)/(n+1) で合成：

 $C = \dfrac{\varepsilon_0 S}{(n+1)\,d'} = \dfrac{\varepsilon_0 S}{d - nt} \quad \left(d' = \frac{d - nt}{n+1}\right)$

R06下問2 / R06上問2 / H30問1 / H18問1 出題。平行平板と違い電界が一様でない（E ∝ 1/r²）ため専用式が必要。レイヤーB — 応用公式

| 公式 | 意味 | 使える条件 | 使えない条件 |
| --- | --- | --- | --- |
| $C = 4\pi\varepsilon_0 a$ | 孤立導体球（半径 a）の静電容量。無限遠を対電極と見なす | 空気中に孤立した導体球・帯電電荷や電位を問う問題 | 対電極が有限距離にある同心球なら次式。「孤立球の容量はゼロ」は誤り（有限値・地球でも約711µF） |

| $E = \dfrac{Q}{4\pi\varepsilon_0 r^2}$ | 内外球間（a≦r≦b）の電界。中心からの距離 r の2乗に反比例 | 球対称・ガウスの法則から導出 | r が一定でない＝電界も一定でない（V=E·d は使えない） |
💡外球を無限遠に飛ばすと孤立球に戻る

同心球の C = 4πε₀·ab/(b−a) で外半径 b→∞ とすると ab/(b−a)→a となり C → 4πε₀a（孤立導体球）に一致。つまり孤立導体球は「内球と無限遠との間のコンデンサ」。
逆に薄い球殻（b−a ≪ a）では a≒b なので S＝4πa²・d＝(b−a) を代入すると C ≒ ε₀·S/d＝平行平板近似が現れる（間隔が半径に比べ十分薄いと曲率が効かない）。

## 7誘電体境界の条件（D と E の連続性）

R05上問1 / H24問15 / H21問17 出題。誘電体の境界を跨ぐとき「何が連続で何が変わるか」を取り違えると失点する。⚠境界で連続な量・変化する量
- 電束密度 D の法線成分（境界面に垂直な向き）は連続（真電荷が無ければ D_n1 ＝ D_n2）。
- 電界 E の接線成分（境界面に沿う向き）は連続（E_t1 ＝ E_t2）。
- その結果、境界を垂直に跨ぐと D は変わらず、E ＝ D/(ε₀ε_r) は誘電率が大きい側で小さくなる。⚠電気力線と電束を混同しない

電気力線の本数は媒質で変わる（ε_r に依存）が、電束（ψ ＝ ε E·S ＝ D·S）は媒質によらず真電荷に等しい。誘電体境界を垂直に跨ぐと E は変わるが D（＝電束）は変わらない、が要点。

| 量 | 境界（法線方向）での挙動 | 媒質依存 |
| --- | --- | --- |
| 電束密度 D ＝ 電束 ψ/S | 法線成分が連続（真電荷なし） | 媒質によらない（真電荷に等しい） |

# コンデンサ

電荷を蓄える受動素子。直流は通さず交流は通す。静電容量・エネルギー・直並列合成が頻出。重要度: S出題頻度: 高難易度: ★★★ホーム›1. 静電気・電子📐公式🧠解説🎯学習💡🧭 コンデンサ問題の解き順（試験中の判定フロー）

「公式は知っているのに解けない」を抜けるための5問判定。順番に答えると公式選定まで一直線。
- 直列か並列か？ → 厚さ方向積層＝直列／面積方向分割＝並列（→ 公式タブ 第3節）
- 電源につながっているか？ → 接続中＝V 一定／切り離し後＝Q 一定（→ 第3節）
- 誘電体・距離・面積の変化があるか？ →  $C = \varepsilon_0 \varepsilon_r S/d$  で C の変化を見る
- Q = CV で整理 → V 一定なら Q が動く／Q 一定なら V が動く
- エネルギー式を選ぶ → V 既知なら  $W = CV^2/2$ ／Q 既知なら  $W = Q^2/(2C)$

## 1原理（なぜ起きるか）🔑直感アナロジー💧コンデンサは電荷を溜める水槽。容量 C [F] が水槽のサイズ、電圧 V [V] が水位、電荷 Q [C] が溜まった水の量。 エネルギーは電界に蓄えられる（電池は化学エネルギー、コンデンサは電界エネルギー）。電流が切れた後も電界が残る。

直流は定常状態で電流ゼロ（水槽が満杯になると水が流れなくなる）。交流では周波数が高いほど容易に電流を通す。

## 2なぜコンデンサで詰まりやすいか（メタ認知）

コンデンサは「公式は知っているのに失点する」典型テーマ。原因は計算ミスではなく認知上の壁。症状と誤モデルを1表で対応付け、セクション3以降を読む前に「自分が今どこで詰まっているか」を意識する。

| 症状（つまずきポイント） | 誤モデル（脳内バグ） | 正モデル（突破の鍵） |
| --- | --- | --- |
| ① 直列／並列で C の式を逆にする | 抵抗と同じ感覚（直列＝足し算）で書いてしまう | 「直列＝極板間が広がる＝容量は小さくなる」と物理イメージで覚える（→ セクション3, セクション4） |
| ② 誘電体挿入で何が変わるか分からない | 「ε_r 倍」を Q/V/W すべてに掛けてしまう | 接続状態で固定される量が違う：定電圧 V固定／定電荷 Q固定（→ セクション3） |
| ③ 直列の各 Q が異なると誤解 | 「直列＝電流分担しない」と混同して Q も分配と思う | 直列＝各 Q は等しい（電荷保存）／分担するのは V（→ セクション4 勘違い②） |
| ④ 極板間引力に F=QE を直接使う | 外部電界の式をそのまま流用 | 極板自身が電界を作るため F=QE/2（自己電界除外・係数1/2が本質）（→ セクション4 勘違い⑤） |
| ⑤ 縦/横誘電体で直列か並列か判別できない | 「2種類入っているから並列」と短絡判断 | 厚さ方向積層＝直列／面積方向分割＝並列（→ 公式タブ 第3節） |
| ⑥ 耐圧計算で容量小コンデンサを軽視 | 「容量大の方が電気量多い→危険」と思い込む | 直列なら Q 等しく V=Q/C → 容量小ほど V 大で先に絶縁破壊（→ セクション4 勘違い④） |

## 3定電圧 vs 定電荷（誘電体挿入後）

誘電体を挿入したとき「何が変わり何が変わらないか」は接続状態で全く異なる。最頻出の引っかけポイント。

| 状態 | 固定されるもの | 変化するもの |
| --- | --- | --- |
| 電源につないだまま（定電圧 V） | 電圧 V（電源が維持） | Q = CV なので Q が ε_r 倍に増加W = CV²/2 なので W が ε_r 倍に増加 |
| 電源を切り離した後（定電荷 Q） | 電荷 Q（逃げ場なし） | V = Q/C なので V が 1/ε_r 倍に減少W = Q²/(2C) なので W が 1/ε_r 倍に減少 |
📍🔑 試験での読み取りポイント

問題文に「電源に接続したまま」「スイッチを閉じたまま」とあれば 定電圧。「電源から切り離した」「スイッチを開いてから」とあれば 定電荷。

> 図：誘電体挿入で C は両方とも εr 倍。定電圧は V 固定なので Q・W が εr 倍に増加、定電荷は Q 固定なので V・W が 1/εr 倍に減少。固定される量が逆だから残りの向きも逆になる。なぜそうなるか（式変形で追う）

共通の出発点: 誘電体を入れると  $C' = \varepsilon_r C_0$ （容量は ε_r 倍に増加）。ここからどう転ぶかは「何が一定か」で分岐する。

電源接続中（V 一定）: 電源が V を保つ →  $Q = CV$  より  $Q' = \varepsilon_r Q_0$ （電荷が増える＝電源から追加供給）。エネルギーは  $W = CV^2/2$  より  $W' = \varepsilon_r W_0$ （増加・電源仕事）。

電源切り離し後（Q 一定）: 外部から電荷出入りなし →  $V = Q/C$  より  $V' = V_0/\varepsilon_r$ （電圧が下がる）。エネルギーは  $W = Q^2/(2C)$  より  $W' = W_0/\varepsilon_r$ （減少・誘電体引き込み仕事を外部にする）。

※「W = CV²/2」と「W = Q²/(2C)」は値は同じだが、定電圧では V 基準・定電荷では Q 基準を使うと一定量が固定されて誤りにくい。

## 4引っかけポイント⚠勘違い①: 直列・並列計算が抵抗と同じだと思いがち

直列のとき合成容量は「逆数の和」（小さくなる）、並列のとき「足し算」（大きくなる）。抵抗と逆転する。「直列は狭くなるから容量小→逆数」で物理的に覚える。⚠勘違い②: 直列接続で各コンデンサの電荷が違うと思いがち

直列接続の各コンデンサの Q は 等しい（電荷保存則）。間の導体は孤立し、総電荷ゼロのまま。電圧分担は  $V_i = Q/C_i$  で計算する。⚠勘違い③: エネルギー公式を混用してミスしがち

 $W = \frac{1}{2}CV^2 = \frac{Q^2}{2C} = \frac{QV}{2}$  はすべて同じ値。与えられているのが V か Q かで使い分けるだけ。1/2 や 2C の係数を忘れない。⚠勘違い④: 直列接続の最大印加電圧で容量小の方を無視しがち

直列接続では各 Q が等しいため、容量の小さい方ほど分担電圧が大きい（V_i = Q/C_i）。耐圧制限に先に達するのは 容量の小さい方。全体最大電圧は「各コンデンサの限界 Q = C_i·V_imax」のうち 最小 Q から逆算する（H28問7）。⚠勘違い⑤: 極板間の引力を F = QE で直接計算（典型ミス）

F = QE は「外部電界中の点電荷」の式・コンデンサ極板には直接使えない。極板自身が電界を作っているため、一方の極板が感じるのは「相手極板が作る電界」だけ。面電荷密度 σ = Q/S として、一枚の極板が作る電界は σ/(2ε₀)。よって  $F = Q \cdot \sigma/(2\varepsilon_0) = \sigma^2 S/(2\varepsilon_0) = QE/2$ （係数 1/2 が本質）。

## 5拡張問い（思考を広げる）📍🧐 なぜ誘電体を入れると ε_r 倍になるのか？（diffuse mode 起動）

誘電体内部の分子が分極し、外部電界を打ち消す方向に内部電界を作る。マクロには「同じ電圧で電荷が ε_r 倍溜まる」=「容量が ε_r 倍」と見える。もう一段深く（電界エネルギー密度と分極）
- 電界エネルギー密度:  $u = \frac{1}{2}\varepsilon E^2$ （J/m³）
- 誘電体内では同じ V で E が同じ（V = Ed）でも、ε が大きいため u が ε_r 倍
- 定電荷状態で誘電体を引き抜くと、引き抜き力に逆らって仕事をする → エネルギーは W₀/ε_r まで減る = エネルギー保存則
- 力学的アナロジー: 誘電体 = 容量を増やす「燃料」・分極 = エネルギーの分配メカニズム← 前1.1 静電気・クーロンの法則次 →1.3 電磁力

## 2例題📊 このセクションの理解度:◎○△×

問1: 4μFと12μFのコンデンサを直列接続したときの合成静電容量を求めよ。解答

直列合成： $\frac{1}{C} = \frac{1}{4} + \frac{1}{12} = \frac{3+1}{12} = \frac{4}{12}$

したがって  $C = 3 \mu\text{F}$

ポイント: コンデンサの直列は抵抗の並列と同じ計算。合成容量は単独の容量より小さくなる。

問2: 容量 C₀ の平行平板コンデンサに比誘電率 ε_r の誘電体を挿入した。電源切り離し後の挿入のとき、エネルギーは何倍になるか？解答

定電荷状態。Q 一定で C = ε_r·C₀。 $W = Q^2/(2C) = Q^2/(2 \varepsilon_r C_0) = W_0 / \varepsilon_r$

したがって 1/ε_r 倍に減る。電源を切り離した後だから、エネルギーは外部仕事（誘電体引き込み）に使われる。

## 8本番5秒チェック（試験トリアージ）📊 このセクションの理解度:◎○△×

本番でコンデンサが出た瞬間に開く3点。知識ではなく「最初に何を見るか」の手順で失点を防ぐ。⚠⏱️ 本番でこのテーマが出たら
- まず何を見る：直列か並列か。誘電体（εr）の有無。電源が接続中(V一定)か切り離し後(Q一定)か。
- 絶対に見落とせない条件：並列＝単純和 C=C1+C2／直列＝逆数和（抵抗と逆＝双対）。C=ε0·εr·S/d。W=½CV²。
- 典型ひっかけ：直並列の合成を抵抗と同じ向きで計算（双対で逆）／電源切り離し後に V 一定と誤認（正しくは Q 一定）。← 前1.1 静電気・クーロンの法則次 →1.3 電磁力
