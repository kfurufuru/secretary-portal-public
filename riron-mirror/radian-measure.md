# 2.6 弧度法

角度を「円弧の長さ÷半径」で表す方式。交流の角速度 ω = 2πf は度ではなくラジアンで定義される。重要度: B（基礎・前提知識）出題頻度: 中（独立出題は稀・交流計算で頻用）難易度: ★★ホーム›2. 電気回路

▼ ミクロを図で掘り下げる：なぜ「弧長＝半径」の角度が 1 rad か

> ミクロ（なぜ）：半径 r と同じ長さの弧を見込む中心角が 1 rad（≈ 57.3°）。角度＝弧長÷半径は長さ同士の比で無次元だから、三角関数の微分・積分で係数 π/180 が現れず、交流の式が最も簡単になる。🔬 ミクロの因果（なぜ 360° ＝ 2π rad か）
- 円に半径 r をとり、円周上に半径と同じ長さ r の弧をとる
- その弧を見込む中心角を 1 rad と定義：θ = ℓ/r（長さ÷長さ＝無次元）
- 1周の弧長は円周 2πr → θ = 2πr ÷ r = 2π rad。つまり 360° = 2π rad（180° = π rad）
- 結果、1秒に f 回転する交流の角速度は  $\omega = 2\pi f$ ［rad/s］— rad だから微分しても係数が出ない

▶ マクロ（解く）は、この下の「2. 公式」の変換表と「3. 主要角の対応表」を参照。試験は  $\theta_\text{rad} = \theta_\text{deg} \times \pi/180$  と  $\omega = 2\pi f$  の運用が全て。

## 1原理（なぜラジアンか）

度数法は「円を360等分」という人為的な単位。これに対しラジアン（弧度法）は 「半径と同じ長さの弧が描く角度」を 1 rad と定義する。半径と弧長が同次元の長さなので比は無次元 — 数学的に自然で、微積分や三角関数の式が最も簡単になる。5秒で思い出す
1周＝ $360^\circ = 2\pi$  rad。半周＝ $180^\circ = \pi$  rad。直角＝ $90^\circ = \pi/2$  rad。

交流では  $\omega = 2\pi f$ （角速度・rad/s）が中心量。位相  $\omega t + \phi$  は時間 t に応じて回るベクトルの角度を表す。度で書くと  $\sin$  の微分等で係数  $\pi/180$  が紛れ込み、式が汚くなる。

## 2公式

| 関係 | 式 | 用途 |
| --- | --- | --- |
| 度→ラジアン | $\theta_\text{rad} = \theta_\text{deg} \times \pi/180$ | 問題文が度で書かれた時の変換 |
| ラジアン→度 | $\theta_\text{deg} = \theta_\text{rad} \times 180/\pi$ | 結果を直感的に確認する時 |
| 角速度 | $\omega = 2\pi f = 2\pi/T$ | 交流回路全般 |
| 瞬時値 | $v(t) = V_m \sin(\omega t + \phi)$ | 正弦波の時間関数表現 |

## 4例題

問題: 周波数 50 Hz の交流の角速度  $\omega$  を求めよ。また、 $t = 5$  ms における位相角を rad と度で示せ。

解答:  $\omega = 2\pi \times 50 = 100\pi \approx 314$  rad/s。
位相角  $\omega t = 100\pi \times 0.005 = 0.5\pi$  rad  $= 90^\circ$ 。
