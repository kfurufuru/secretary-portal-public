# G 複素数・フェーザ演算

交流計算の共通言語。j は「90°回転」の演算子で、波の足し算掛け算を「矢印（フェーザ）の演算」に変える。ここの計算が即答できると 2.2〜2.4・4.1 の計算速度が一段上がる。対象: STEP 4（交流回路基礎）に入る前・交流計算で詰まったとき狙い: j計算と極形式⇄直交形式の変換を反射化ホーム›G 複素数・フェーザ演算📍なぜ複素数を使うか（30秒で）交流をそのまま扱うと sin・cos の加法定理だらけになる。正弦波を「大きさと角度を持つ矢印（フェーザ）」＝複素数に置き換えると、波の合成がベクトルの足し算に、微分が ×jω に、積分が ÷jω に化ける。つまり複素数は計算を楽にするための道具。道具の操作（この頁）さえ反射化すれば、交流回路は直流回路と同じ「オームの法則の世界」になる。

## 1j の正体 ＝「90°回転」の演算子

 $j^2 = -1$  は「90°回転を2回すると 180°回転（符号反転）になる」という意味。j を掛けるたびに矢印が反時計回りに 90° 回る。

| 演算 | 結果 | 幾何の意味 |
| --- | --- | --- |
| 1 × j | j | +90°回転 |
| j × j | −1 | +180°回転（反転） |
| −1 × j | −j | +270°回転 |
| −j × j | 1 | 一周して元に戻る |
| 1 ÷ j | −j | −90°回転（1/j = −j） |
💡電験での j の顔コイル  $Z_L = j\omega L$ （電圧が電流より 90°進む＝電流が遅れる）／コンデンサ  $Z_C = \dfrac{1}{j\omega C} = -j\dfrac{1}{\omega C}$ （電流が 90°進む）。2.2 交流の基礎の ELI / ICE はこの j の符号のこと。

## 2直交形式 ⇄ 極形式の変換

同じ複素数の2つの書き方。直交形式  $a + jb$ （実部・虚部）と極形式  $r\angle\theta$ （大きさ・角度）。

 $r = \sqrt{a^2 + b^2},\quad \theta = \tan^{-1}\dfrac{b}{a} \qquad\Longleftrightarrow\qquad a = r\cos\theta,\quad b = r\sin\theta$

試験で出る三角形はほぼ次の3つ。数値を見た瞬間に角度が出るまで覚える：

| 辺の比（実部 : 虚部 : 大きさ） | 角度 | 出る場面の例 |
| --- | --- | --- |
| 3 : 4 : 5（4 : 3 : 5） | 53.1°（36.9°） | R=3, X=4 → Z=5∠53.1°・力率 0.6（0.8） |
| 1 : 1 : √2 | 45° | R = X のとき・力率 0.707 |
| 1 : √3 : 2（√3 : 1 : 2） | 60°（30°） | 三相の 30°・力率 0.5（0.866） |

## 3四則演算は「形式の使い分け」が9割

| 演算 | 使う形式 | ルール | 例 |
| --- | --- | --- | --- |
| 足し算・引き算 | 直交形式 | 実部同士・虚部同士を足す | (3+j4) + (1−j2) = 4+j2 |
| 掛け算 | 極形式 | 大きさは掛ける・角度は足す | 5∠53.1° × 2∠30° = 10∠83.1° |
| 割り算 | 極形式 | 大きさは割る・角度は引く | 8∠50° ÷ 2∠20° = 4∠30° |

形式が混在していたら先に変換してから演算する（足すなら直交へ・掛けるなら極へ）。「直交形式のまま掛け算」も展開すればできるが、本番では極形式の方が速く・ミスが少ない。

## 4共役と有理化 — 分母の j を消す

 $Z = a + jb$  の共役は  $\bar{Z} = a - jb$ （虚部の符号を反転）。積は実数になる： $Z\bar{Z} = a^2 + b^2$ 。これで分母の j を消す：

 $\dfrac{1}{3+j4} = \dfrac{3-j4}{(3+j4)(3-j4)} = \dfrac{3-j4}{25} = 0.12 - j0.16$
- 使いどころ①：並列回路のアドミタンス  $Y = 1/Z$ （→ F 双対マップ の Z↔Y）
- 使いどころ②：複素電力  $S = V\,\bar{I}$ （電流の共役を掛ける。実部が有効電力 P・虚部が無効電力 Q）

## 7引っかけ3点⚠複素数計算で落ちる典型3点
- 角度の符号と遅れ/進み：誘導性（コイル優勢）の負荷では電流は電圧より遅れ →  $\dot{I} = I\angle(-\theta)$ 。「j が付く＝遅れ」ではなく、Z の偏角 +θ が I では −θ になる（割り算で角度を引くから）。
- tan⁻¹ の象限：電卓の tan⁻¹ は −90°〜+90° しか返さない。実部が負の複素数（180°側）では±180°の補正が要る。3種の回路計算ではほぼ第1・第4象限だが、虚部の符号（X_L−X_C の正負）は毎回確認。
- 度とラジアンの混在：ωt は rad、選択肢は度のことが多い。電卓の角度モード（DEG/RAD）を解き始めに確認（→ 2.6 弧度法）。
