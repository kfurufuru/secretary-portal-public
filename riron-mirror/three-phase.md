▼ ミクロを図で掘り下げる：なぜ帰り線なしの3本で送れるか（平衡三相）

> ミクロ（なぜ）：平衡三相では、120°ずつずれた3相の瞬時値がどの瞬間も打ち消し合い、和が常に 0。だから中性線に電流が流れず、帰り線を省略して3本の電線で送れる。不平衡になるとこの前提が崩れ、中性線に電流が流れる。🔬 ミクロの因果（なぜ3本の線で送れるか・平衡三相）
- 発電機の3組のコイルを120°ずつずらして配置 → 等しい振幅・120°位相差の3相電圧が生まれる
- 平衡負荷なら3相の電流も等大・120°ずれの正弦波になる
- どの瞬間も3つの瞬時値は＋と−が打ち消し合い、和が常に 0（例：+0.87 −0.87 +0）
- 結果、中性線電流 0 → 帰り線を省略し3本で送電。瞬時電力の合計も一定（単相のような脈動がない）

三相交流は 120°ずつずらした3つの単相の組。電圧の測り方が2通りあり、相電圧（各相と中性点の間）と線間電圧（電線どうしの間）を区別する。Y結線では：

 $V_{\text{線間}} = \sqrt{3}\,V_{\text{相}}, \qquad P = \sqrt{3}\,V_{\text{線間}}\,I_{\text{線}}\cos\varphi\;[\mathrm{W}]$

√3 ≈ 1.73 は暗記値ではない：大きさが等しく120°ずれた2つのベクトルの差を取ると、余弦定理から大きさが √3 倍（向きは30°ずれ）になる、という幾何学の必然。Δ結線では電圧側が一致し電流側に √3 が移る（I_線 = √3 I_相）。

因果でつなぐと 120°対称 → 3相の瞬時電力の脈動が打ち消し合い合計が一定 → 同じ電力を細い3本線で送れる。これが三相が送電の標準である理由。Y か Δ かで「√3 が電圧と電流のどちらに付くか」が変わるのが最頻出の引っかけ。🧩 チャンク階層 — どこを覚えているか自己確認第1層V / I線間 vs 相 — Y は電圧に √3、Δ は電流に √3第2層P三相電力 — P = √3 VL IL cosφ（線間で書くか相で書くか）第3層Z_Y ↔ Z_ΔY-Δ 等価変換 — Z_Y = Z_Δ / 3（平衡三相のみ）自分が今どの層で迷っているか自己診断 → 該当公式へ → 例題で確認、の順で進める。💡⚡ 5秒で思い出す

三相 ＝ 120°ずらした3相。3本線で単相の √3 倍を運ぶ送電方式。√3 は「120°ずれた等しいベクトルの差」から必然的に出る（余弦定理）。
- 平衡三相：位相 120° ずれた3相が対称（合成 = 0、中性線電流 = 0）
- Y結線：VL = √3 × VP ／ IL = IP（電圧に √3）
- Δ結線：VL = VP ／ IL = √3 × IP（電流に √3）
- 三相電力：P = √3 × VL × IL × cosφ（線間で書くと √3 が必須）
- Y↔Δ 変換：Z_Y = Z_Δ / 3（平衡のみ・Y側が小さい）

## 6不平衡負荷と力率改善（問15対策）📊 このセクションの理解度:◎○△×📝不平衡Y負荷（中性線あり4線式）の解き方 — R04上問15

各相のインピーダンスが違うと「1相分×3」は使えない。中性線がある4線式なら各相を独立した単相回路として解ける。
- 各相の相電圧は電源から直接（ $V_P = V_L/\sqrt{3}$ ）
- 相ごとに  $I_a=\dfrac{V_a}{Z_a},\;\; I_b=\dfrac{V_b}{Z_b},\;\; I_c=\dfrac{V_c}{Z_c}$
- 中性線電流はキルヒホッフ電流則のベクトル和  $\dot I_n=\dot I_a+\dot I_b+\dot I_c$ （平衡なら 0）
- 不平衡の三相電力は各相の算術和  $P=P_a+P_b+P_c$

 $P=\sqrt{3}\,V_L I_L \cos\varphi$  は平衡三相専用。不平衡には使えない。✅正解の原則相電圧＝線間電圧÷√3 が成立するのは中性点が固定されている場合（中性線あり4線式、または平衡負荷）のみ。3線式の不平衡は中性点電位がシフトするので、ミルマンの定理で中性点電位 V_N を求め、各相の負荷電圧＝電源の相電圧（E_a等）−V_N として計算する。⚠よくある誤解三相3線式の不平衡回路でも、各相電圧は線間電圧を√3で割れば求まると思う。🔍判別ステップ
- 中性線の有無を確認（4線式か3線式か）
- 中性線あり4線式 or 平衡 → 各相電圧＝V_L/√3 が使える
- 3線式の不平衡 → ミルマンの定理で中性点電位 V_N を求め、各相の負荷電圧＝電源の相電圧（E_a等）−V_N
- 電験3種の不平衡は中性線ありの4線式で出ることが多い（3線式不平衡は連立が必要で稀）出典: パターン31・H27問8・R04上問9📝三相の力率改善（並列コンデンサ）— R06下問15

誘導性の平衡負荷に並列にコンデンサを入れて力率1（線電流最小）にする C を求める頻出型。単相の力率改善公式をそのまま流用しない（√3・Y/Δ が絡む）。三相無効電力どうしを釣り合わせるのが最短。 $Q_{load}=\sqrt{3}\,V_L I_L \sin\varphi$  $Q_C = 3\,\omega C V_P^2 = 3\,\omega C \left(\dfrac{V_L}{\sqrt{3}}\right)^2 = \omega C V_L^2 \quad(\text{Y接続3個})$  $Q_C=Q_{load}\;\Rightarrow\; C=\dfrac{Q_{load}}{\omega V_L^2}$

相電流ベースでも同じ：遅れ無効分  $I_Q=I_P\sin\varphi$ 、Y接続 C の進み電流  $I_C=\omega C V_P$  を等しくして  $C=\dfrac{I_Q}{\omega V_P}$ 。⚠Δ接続コンデンサは Y接続の3倍効果

| 構造 | 各相が中性点で1点に集まる（星形） | 各相が三角形ループに繋がる |
| VL と VP | VL = √3 VP（差を取るので √3） | VL = VP（相が線間に直結） |

| 得意分野 | 低圧側・接地系統・配電 | 高圧側・電動機巻線・送電 |
💡覚え方

「Y は電圧で√3、Δ は電流で√3」と覚える。理由は セクション4「線間-相の導出」で。両者とも三相電力は  $P = \sqrt{3} V_L I_L \cos\varphi$  の同じ形になる（VL と IL で書けば結線によらず共通）。

## 4線間-相 関係の導出（なぜ √3 が出るか）📊 このセクションの理解度:◎○△×

三相交流で最も「暗記で終わらせがち」なのが √3 倍の関係。ここではベクトル図と余弦定理で必然性を示す。

### Y結線：VL = √3 VP の理由

Y結線の各相電圧は中性点 N から出る等大ベクトル（位相 0°, −120°, +120°）。線間電圧 V_ab は2 つの相電圧の差：

 $V_{ab} = V_a - V_b$

120° ずれた等大ベクトルの差の大きさを余弦定理で計算：

 $|V_{ab}|^2 = V_P^2 + V_P^2 - 2V_P^2 \cos(120°) = 2V_P^2 + V_P^2 = 3V_P^2$

 $\therefore |V_{ab}| = \sqrt{3}\,V_P$

これが √3 の正体。120° ずれた 2 つの等大ベクトルの差から必然的に出る。暗記ではない。

> 余弦定理のステップ図解：Va − Vb の合成

> ① Va (上) と Vb (右下・120°差) → ② Vb を反転して −Vb (Va と 60° の角度) → ③ Va と −Vb の合成が Vab。|Vab| = √3 × Vp（120°差の余弦定理が必然）。💡👁️ 余弦定理の腑落ち（監修者 input 反映・2026-05-29）

監修者さん指摘 (PR #85): 「√3 の余弦定理導出が暗記止まり」。上のステップ図解で 「なぜ √3 が必然か」 を視覚的に確認:
- Va と Vb は 120° 開いた等大ベクトル（どちらも長さ Vp）
- −Vb は Vb の反対方向（長さ Vp のまま・180° 回転）
- Va と −Vb のなす角は 60°（180°−120°＝60°）
- 余弦定理:  $|V_{ab}|^2 = V_p^2 + V_p^2 + 2 V_p^2 \cos(60°) = 2V_p^2 + V_p^2 = 3V_p^2$  ⇒ 結局  $3 V_p^2$
- ∴ |Vab| = √3 Vp — 120° 等大ベクトル差は 「必ず √3 倍」

※ Va + (−Vb) = Va − Vb と等価。図ではベクトル合成として「Va 先端から −Vb 方向」に並べた。

### Δ結線：IL = √3 IP の理由

Δ結線の各相電流はループ内を流れる。線電流 I_a は隣接する 2 つの相電流の差：

 $I_a = I_{ab} - I_{ca}$

同じく 120° ずれた等大ベクトルの差なので、全く同じ余弦定理計算で：

 $|I_a| = \sqrt{3}\,I_P$ 💡本質

Y も Δ も √3 の出る理由は同じ（120° ベクトル差）。違うのは「差を取る対象」が電圧か電流か。Y は中性点があるから相電圧 ⇒ 線間電圧は電圧差。Δ は電源側が線間に直結し、相電流がループを流れるから線電流は電流差。

## 5三相電力 P = √3 VL IL cosφ の導出📊 このセクションの理解度:◎○△×

平衡三相の有効電力は単純に「1 相分の電力 × 3」：

 $P = 3\,V_P I_P \cos\varphi$

これを線間電圧・線電流で書き換える。Y結線の場合、 $V_P = V_L/\sqrt{3}$ 、 $I_P = I_L$  を代入：

 $P = 3 \cdot \frac{V_L}{\sqrt{3}} \cdot I_L \cdot \cos\varphi = \sqrt{3}\,V_L I_L \cos\varphi$

Δ結線でも  $V_P = V_L$ 、 $I_P = I_L/\sqrt{3}$  で同じ結果になる。つまり線間で書けば結線によらず共通形。⚠cosφ の適用条件

cosφ は負荷側の力率（電圧と電流の位相差の余弦）。系統全体の力率と混同しない。問題文で「力率 cosφ = 0.8 遅れ」と与えられたら、その負荷の電圧・電流の位相差から決まる値。

## 6Y-Δ等価変換の使いどころ📊 このセクションの理解度:◎○△×

平衡三相負荷の Y-Δ 等価変換は係数 3 倍の単純式：

 $Z_Y = \frac{Z_\Delta}{3} \quad \Leftrightarrow \quad Z_\Delta = 3\,Z_Y$

使いどころは「電源 Y・負荷 Δ」または「電源 Δ・負荷 Y」の混在回路。Δ 負荷を Y 等価に変換すれば Y-Y 形になり、1 相分の等価回路で解ける。💡覚え方

Y 側が小さい（÷3）。Δ は 3 個並列に見えるので合成抵抗が大きく見えるイメージ。逆向きは ×3。「Y → Δ なら ×3、Δ → Y なら ÷3」と方向で覚える。⚠不平衡では使えない

Z_Y = Z_Δ / 3 は平衡三相専用。各相のインピーダンスが異なる不平衡負荷には一般の Y-Δ 変換式（3 抵抗の積/和）が必要だが、電験 3 種ではほぼ出ない。📍🧑 2 電力計法の原理（Blondel定理→公式導出・監修者 input 反映 2026-05-29）

監修者さん指摘 (PR #85): 「2電力計法は理解が足りない・公式機械暗記」。Blondel 定理から導出すれば「なぜ 2 つで足りるか」「なぜ √3 が出るか」が一本道で繋がる。

① Blondel 定理: n 本の電線がある回路の総電力は、(n−1) 個の電力計で測定できる。三相 3 線式 (n=3) なら 2 個の電力計で完全測定可能。

② 接続方法: 線 c を電圧基準点として、W₁ は Vac × Ia、W₂ は Vbc × Ib を測る（c 線には電力計を入れない）。

 $W_1 = V_{ac} I_a \cos\theta_1, \quad W_2 = V_{bc} I_b \cos\theta_2$

③ 有効電力の和 P = W₁ + W₂ の導出: 瞬時電力  $p_a + p_b + p_c$  を c 基準で書き換えると、c の項がキルヒホッフ電流則  $i_a + i_b + i_c = 0$  で消える:

 $P = W_1 + W_2 = \sqrt{3}\,V_L I_L \cos\varphi$

④ 力率角の導出 tan φ = √3(W₁−W₂)/(W₁+W₂): 平衡三相では位相関係から:

 $W_1 - W_2 = V_L I_L \sin\varphi$

これを  $W_1 + W_2 = \sqrt{3} V_L I_L \cos\varphi$  で割ると:

 $\frac{W_1 - W_2}{W_1 + W_2} = \frac{\tan\varphi}{\sqrt{3}} \;\;\Rightarrow\;\; \tan\varphi = \frac{\sqrt{3}(W_1 - W_2)}{W_1 + W_2}$

特殊ケース表（暗記回避用）:

| 力率 cosφ | W₁ と W₂ の関係 | 解釈 |
| --- | --- | --- |
| 1.0（純抵抗） | W₁ = W₂ | tan φ = 0・両電力計が等しく振れる |
| 0.5（60° 遅れ） | W₂ = 0 | tan φ = √3・W₂ がゼロ |
| < 0.5 | W₂ < 0 | tan φ > √3・W₂ が逆振れ → 端子を入替え負値として代入 |

※ 監修者さんの暗記回避方針: tan φ の公式は 「sin/cos = (W₁−W₂)/(W₁+W₂) を √3 倍」と関係式で記憶する（公式そのものを丸暗記しない）。

## 7引っかけポイント（よくある勘違い）📊 このセクションの理解度:◎○△×⚠① Y結線で IL = √3 IP としてしまう（Δ と Y の混同）

Y結線は  $I_L = I_P$ 。√3 が出るのは Δ 結線の電流と Y 結線の電圧。「Y は電圧、Δ は電流」と覚える。⚠② 三相電力を相表記 3V_P IP と線間表記 √3 VL IL で混在

線間表記なら √3、相表記なら 3。両者を混ぜて P = 3 VL IL cosφ と書くと値が √3 倍ズレる。式に入れる V と I がどちら表記かを毎回確認。⚠③ Y-Δ 変換で 1/3 と 3 を逆にする

Y 側が小さい：Z_Y = Z_Δ / 3。Δ → Y なら÷3、Y → Δ なら×3。方向で覚えて係数を間違えない。⚠④ 不平衡負荷で「1 相分×3」で計算（R04上 問15 型）

1 相分×3 が使えるのは平衡三相のみ。不平衡では各相を個別に計算してベクトル合成。中性線電流も 0 にならない。⚠⑤ 電源Δ・負荷Y の混在で「Δ電源だから電圧に √3」と誤適用

線間電圧は結線によらず共通の基準。√3 が入るのはY 結線の相電圧を出すとき（÷√3）だけ。Δ 電源 → 線間電圧そのまま → Y 負荷の相電圧は VL / √3。⚠⑥ 2 電力計法で W₂ < 0 のとき正のまま計算（R02 関連）

力率 < 0.5 のとき電力計が逆振れ。「逆接続で測定した」と認識し、W₂ を負値として公式に代入する。 $P = W_1 + W_2$  のとき W₂ = −|W₂|。tanφ も  $\tan\varphi = \sqrt{3}(W_1-W_2)/(W_1+W_2)$  で W₂ < 0 を反映。

- ⑤ (b) Δ電源の相電流 İ_ab：Δ では I_線 = √3 I_相、線電流は相電流より 30°遅れる。∴ İ_ab = (İ_1/√3)∠(+π/6) = (11.55/√3)∠(−π/3 + π/6) = 6.67∠(−π/6) A → 選択肢(2)
- ⑥ 検算：負荷の相電流 11.55 A に対し Δ電源の相電流は 11.55/√3 ≈ 6.67 A（大きさ比 √3・位相 30°回転）で整合。本問では電力 P は問われていない📝 まず自分で解答 → 👁 正解・解説を見る過去問は常に即表示にする⚠本問のひっかけポイント（5項目）
- Δ電源の「相電圧」＝線間電圧: Δ結線は相電圧と線間電圧が等しい（V_L = 200 V をそのまま使う）。「電源だから √3 倍／除」は誤り。Y電源で相電圧 200 V なら線間 200√3 V となり別問題
- Y負荷の相電圧は V_L/√3: 混在型では必ずこの 1 段を挟む（200/√3 ≈ 115.5 V）。Δ負荷なら相電圧＝線間電圧でこの段は不要
- 30°の位相ずれを落とさない: Y結線で相電圧は線間電圧より 30°遅れ、Δ結線で線電流は相電流より 30°遅れる。極形式の角度はこの ±π/6 の積み重ね

- |Z| と偏角を混同しない: |Z| = √((5√3)² + 5²) = 10 Ω、偏角 θ = arctan(1/√3) = π/6。力率 cosθ = 5√3/10 = √3/2 ≈ 0.866（≒30°）

## 43条件P即答チェック（限界的練習ゾーン）📊 このセクションの理解度:◎○△×

「解ける／解けない」の境界を強制発火させる練習。同一パラメータ（VL = 200 V, IL = 10 A, Y結線平衡負荷）× 力率 3 条件で三相有効電力を即答し、典型誤答 3 択でセルフ判定。3 条件すべて正答できれば B 問題対応 OK の目安。📍共通設定

線間電圧 VL = 200 V・線電流 IL = 10 A・Y結線平衡負荷。力率 cosφ を 3 条件で変える。 $P = \sqrt{3}\,V_L I_L \cos\varphi$  を即答。

即答してから下の解答を開くこと（暗算 30 秒目安・√3 ≈ 1.732）。

| 条件 | cosφ | 三相有効電力 P [W] | 用途イメージ |
| --- | --- | --- | --- |
| ① 純抵抗負荷 | 1.0 | √3 × 200 × 10 × 1.0 ≈ 3464 W | 電気ヒータ・抵抗炉 |
| ② 一般的な誘導性負荷 | 0.8（遅れ） | √3 × 200 × 10 × 0.8 ≈ 2771 W | 標準的な三相モータ |
| ③ 低力率（軽負荷モータ等） | 0.6（遅れ） | √3 × 200 × 10 × 0.6 ≈ 2078 W | 軽負荷モータ・力率改善前 |
解答の導出（3条件まとめて）

① cosφ = 1.0: P = √3 × 200 × 10 × 1.0 = 2000√3 ≈ 3464 W。皮相と一致（S = P・Q = 0）

② cosφ = 0.8: P = √3 × 200 × 10 × 0.8 = 1600√3 ≈ 2771 W。Q = √3 × 200 × 10 × 0.6 ≈ 2078 var

③ cosφ = 0.6: P = √3 × 200 × 10 × 0.6 = 1200√3 ≈ 2078 W。Q = √3 × 200 × 10 × 0.8 ≈ 2771 var（P と Q が逆転）

※ P は cosφ に比例。力率が 1.0 → 0.6 で電力が 60% に減る。力率改善（cosφ を 1 に近づける）の重要性が数値で見える。⚠典型誤答3択（自己採点用）
- √3 を忘れて P = VL IL cosφ（誤）⇒ 正解は  $\sqrt{3}\,V_L I_L \cos\varphi$ 。線間表記なら √3 必須
- √3 を 2 倍にして P = 2 VL IL cosφ（誤）⇒ √3 ≈ 1.732 で 2 倍ではない。120° ベクトル差から出る係数
- cosφ を sinφ と取り違える（誤）⇒ 有効電力は cosφ、無効電力が sinφ。問題文の「力率」は cosφ💡セルフ判定

3 条件すべて即答 ⇒ B 問題（三相電力＋Y/Δ 混在）でも止まらない。1 問でも誤答 ⇒ 解説タブ セクション5 三相電力導出で復習。

## 9本番5秒チェック（試験トリアージ）📊 このセクションの理解度:◎○△×
