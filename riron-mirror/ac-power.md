使う記号: V（電源電圧・V）／I（電流・A）／φ（電圧と電流の位相差・rad or °）／cosφ（力率・無次元・0〜1）／sinφ（無効率・無次元）／P（有効電力・W）／Q（無効電力・var）／S（皮相電力・VA）／QC（補償無効電力・var）／C（進相コンデンサ容量・F）／R（抵抗・Ω）／XL=ωL（誘導リアクタンス）／XC=1/(ωC)（容量リアクタンス）／Z（インピーダンス・Ω）／ω=2πf（角周波数・rad/s）📝📘 有効・無効・皮相電力 P／Q／S とは（定義）

交流では電圧と電流に位相差 φ があるため、「電力」が3つに分かれる。電流を 電圧と同位相な成分と90°ずれた成分に分解したものがそれぞれ P と Q：

 $P = VI\cos\varphi\;[\mathrm{W}],\quad Q = VI\sin\varphi\;[\mathrm{var}],\quad S = VI\;[\mathrm{V\!A}],\quad S^2 = P^2 + Q^2$

単位は3つとも「電圧×電流」の次元だが役割で使い分ける：W（仕事になる分）／var（L↔C を往復するだけの分）／VA（設備が受け持つ見かけの容量）。cosφ が 力率（S のうち仕事になる割合）。

因果でつなぐと cosφ 小（位相ずれ大）→ 同じ P でも Q が大きい → S = √(P²+Q²) が膨らむ → 設備容量・契約料金が増える。P/Q/S は別々の電力ではなく 同じ電流の直角分解で、幾何学的には電力三角形（直角三角形）の3辺。コンデンサで Q を打ち消して S を縮めるのが力率改善。🧩 チャンク階層 — どこを覚えているか自己確認第1層V · I · cosφ電圧・電流・位相差 — cosφは「電流のうち電圧と同位相な成分の割合」第2層P · Q · S3成分 — P=VIcosφ・Q=VIsinφ・S=VI・S² = P² + Q²（直角三角形）第3層QC → C力率改善 — QC = P(tanφ₁−tanφ₂)・C = QC/(ωV²)第1層（V・I・cosφ）から第2層（電力3成分）が出て、第3層（力率改善）が実務応用。迷ったら第1層（電流の位相分解）に戻る。💡⚡ 5秒で思い出す

交流電力 ＝ 電流を電圧と同位相な成分と90°ずれた成分に分解した両方を「電力」と呼ぶ。同位相分が仕事（P）・90°ずれ分は往復するだけ（Q）・両者の幾何合成が設備容量（S）。
- 有効電力：P = VI cosφ [W]・抵抗で消費される実エネルギー
- 無効電力：Q = VI sinφ [var]・LとC間でエネルギー往復・正味消費ゼロ

📍📐 力率改善公式 Q_C = P(tanφ₁−tanφ₂) の導出

改善前の無効電力 Q₁ = P tanφ₁（P=VIcosφ₁・Q₁=VIsinφ₁ より Q₁/P = sin/cos = tan）。改善後 Q₂ = P tanφ₂。P は改善前後で不変（コンデンサは無効電力のみ補償・有効電力には触らない）なので、必要な補償 Q_C = Q₁ − Q₂ = P(tanφ₁ − tanφ₂)。「P が掛かる」のは「P を変えずに Q だけ減らす」操作だから。

## 1原理：仕事 vs エネルギー往復🧠 脳に残す一言交流電力 ＝ 電流を「電圧と同位相な成分」と「90°ずれた成分」に分解した両方を電力と呼ぶ。同位相分は仕事に変わる（P）・90°ずれ分は L↔C を往復するだけで消費ゼロ（Q）・両者の幾何合成 √(P² + Q²) が設備容量（S）。🔑直感アナロジー⚙️川の流れに見立てると、水車を回す流れ（有効）と、岸に向かって行ったり来たりするだけの揺れ（無効）がある。両方が同じ川幅（皮相）を占めるが、仕事をするのは前者のみ。後者は無駄に岸を侵食するだけ。

工場の電気代は使用エネルギー量（P×時間=kWh）と契約設備容量（S=kVA）の両方で決まる。Q が大きいと S が膨らみ契約料金が上がるため、コンデンサで Q を打ち消して S を縮める「力率改善」が実務技術。

## 3P・Q・S の物理的意味

P（有効電力）は瞬時電力  $p(t) = v(t) \cdot i(t)$  の時間平均。電圧  $v(t) = V\sqrt{2}\sin\omega t$ ・電流  $i(t) = I\sqrt{2}\sin(\omega t - \varphi)$  のとき：

 $P = \langle v(t) i(t) \rangle = VI\cos\varphi \quad [\text{W}]$

Q（無効電力）は瞬時電力の振動成分の振幅。電流の sinφ 成分（電圧と90°ずれ）が L↔C 間で往復するだけ：

 $Q = VI\sin\varphi \quad [\text{var}]$

S（皮相電力）は単純な電圧・電流の積。P と Q を幾何合成した斜辺：

 $S = VI = \sqrt{P^2 + Q^2} \quad [\text{VA}]$ ⚠単位が違う理由

物理次元はすべて [J/s] で同じ。しかし「消費される」か「往復する」か「設備容量か」で意味が違うため、W / var / VA と別単位で書き分ける。これは IEC 規格（国際的取り決め）で、混同すると工場の契約・設計が破綻するため徹底される。

> 図：電力三角形。P＝消費（W）、Q＝往復するだけの無効分（var）、S＝合成＝√(P²+Q²)（VA）。直角三角形なので P と Q は単純に足せない。力率改善はコンデンサで Q を打ち消し（QC）、S を S' に縮めて cosφ=P/S を上げること。

## 4力率 cosφ の4視点まとめ💡🧐 同じ cosφ を4つの異なる側面から見る（監修者自己観察③直接対応）

力率 cosφ は同じ無次元量を4つの違う側面で表現している。1つでも腑に落ちれば全て繋がる（rlc-resonance Q値4視点と同型）。

| 視点 | 表現 | 直感イメージ |
| --- | --- | --- |
| ① 幾何（インピーダンス三角形） | $\cos\varphi = R / Z$ | 抵抗 R と全インピーダンス Z の比。Z = √(R² + X²) の隣辺/斜辺 |
| ② 電力（電力三角形） | $\cos\varphi = P / S$ | 皮相電力のうち実際に仕事をする割合。S² = P² + Q² の隣辺/斜辺 |
| ③ 位相（電流分解） | $\cos\varphi = I_{\text{有効}} / I$ | 電流ベクトルのうち電圧と同位相な成分の割合。電流を電圧軸に射影した長さ÷電流大きさ |
| ④ 時間平均（瞬時電力） | $\cos\varphi = \langle v(t) i(t) \rangle / (VI)$ | 瞬時電力の時間平均 ÷ 電圧電流実効値の積。1秒間に運ばれた純エネルギー比 |

本質を 1 行で: cosφ ＝ 「電流のうち電圧と同位相な成分の割合」。幾何・電力・位相・時間平均のどれで測っても同じ値。力率改善とは「電流分解で sin成分（直交）を C で打ち消し、cos成分（同位相）の比率を上げる」こと。

※ 試験問題は4視点のどれかで聞いてくる。「電力比 cosφ=P/S」だけ覚えると、インピーダンス比 R/Z で出題されたとき公式が出てこない。

## 5力率改善：原理・計算手順・過補償

誘導性負荷（モータ等）に並列にコンデンサを接続すると、Cの進相無効電力で負荷の遅相無効電力を打ち消す。P には触らず Q だけ減らすことで、S = √(P² + Q²) が小さくなる ⇒ 契約 kVA が下がる ⇒ 電気代が下がる。📍計算手順（4ステップ）

1. 改善前の無効電力： $Q_1 = P\tan\varphi_1$

2. 改善後の無効電力： $Q_2 = P\tan\varphi_2$ （目標力率 cosφ₂ から tanφ₂ = sinφ₂/cosφ₂）

3. 必要な補償容量： $Q_C = Q_1 - Q_2 = P(\tan\varphi_1 - \tan\varphi_2)$  [var]

4. コンデンサ容量： $C = \dfrac{Q_C}{\omega V^2}$  [F]⚠過補償（H22 問8 頻出）

Q_C を Q₁ より大きく取ると Q₂ < 0 となり、進相力率（容量性）に振れる。「力率を改善したつもりが逆方向に行った」という引っかけ。目標 cosφ₂ = 1（Q₂ = 0） が最も計算しやすい目安。実務では cosφ = 0.95〜0.99 程度を目標にする（完全に1にすると共振リスク）。

## 6ひずみ波・RC直列・周波数特性（R06上・R04下・H19 頻出）

### ひずみ波の電力（R06上 問9）

高調波を含む交流（ひずみ波）の有効電力は周波数ごとに個別計算して合算する：

 $P = V_1 I_1 \cos\varphi_1 + V_3 I_3 \cos\varphi_3 + V_5 I_5 \cos\varphi_5 + \cdots$

実効値は  $I = \sqrt{I_1^2 + I_3^2 + I_5^2 + \cdots}$ ・ $V = \sqrt{V_1^2 + V_3^2 + \cdots}$ 。異なる周波数成分間では平均電力の積分がゼロ（直交）なので個別合算でよい。P = VI cosφ（基本波の値だけ）はひずみ波には使えない。

### RC直列回路の消費電力（R04下 問9）

コンデンサは電力を消費しない。有効電力は抵抗 R のみ：

 $Z = \sqrt{R^2 + X_C^2}, \quad I = V/Z, \quad P = I^2 R = \dfrac{V^2 R}{R^2 + X_C^2}$

P = V²/R を使ってはいけない：この式は「電源電圧 V が全部 R にかかる」純抵抗回路のみ。直列では V_R < V（R と X_C で分圧）。

### 周波数変化と力率（H19 問8）

| 回路 | 周波数 ↑ のとき | 力率の変化 |
| --- | --- | --- |
| RL 直列 | X_L = ωL ↑ → Z ↑ | cosφ = R/Z 減少（力率悪化） |
| RC 直列 | X_C = 1/(ωC) ↓ → Z ↓ | cosφ = R/Z 増加（力率改善） |

共振条件（RLC直列・X_L = X_C）では cosφ = 1（最大）。この周波数が共振周波数 f₀ = 1/(2π√(LC))。

## 7よくある勘違い（引っかけ集 7項目）⚠① 無効電力の単位を [W] と書く

❌ 単位を全部 [W] にしてしまう。✅ P=W・Q=var・S=VA。3つとも物理次元は J/s で同じだが、意味が違うため単位を書き分ける（IEC 規格）。試験では選択肢に単位違いの罠が混入する。⚠② 力率計算で cosφ と sinφ を逆に使う

❌ 「P = VI sinφ」「Q = VI cosφ」と書く。✅ P = VI cosφ（コサインが有効・コ＝ショウヒ）・Q = VI sinφ。電流ベクトルを電圧軸に射影した cos成分が「仕事をする部分」と覚える。⚠③ P + Q = S と足し算する（パターン29）

❌ 「3つとも電力なら足せる」。✅ P と Q は直交ベクトル。S² = P² + Q²（電力三角形・S が斜辺）。Q の符号は負荷で変わる（誘導性で正・容量性で負）。符号を誤ると力率改善の補償方向が逆になる致命ミス。⚠④ 単相3線式の消費電力を「電源電圧 × 2」で計算（H22 問7 類出）

❌ 線間 200V × 2 で計算する。✅ 単相3線式は中性線を共有する2つの単相回路の組合せ。各負荷が線間電圧（200V）か中性線間電圧（100V）のどちらに接続されているかを回路図で確認。平衡負荷なら中性線電流ゼロ。消費電力はP = I₁²R₁ + I₂²R₂ で合算。⚠⑤ 過補償で進相側に振れる（H22 問8 類出）

❌ コンデンサ容量を大きくすれば力率は1に近づくと思い込む。✅ Q_C > Q₁ になると Q₂ < 0（進相側）。「力率改善したのに方向が逆」になる。目標 Q₂ = 0（cosφ₂ = 1） が最も安全。⚠⑥ RC直列で P = V²/R を使う（R04下 問9）

❌ 「P = V²/R は公式だから使える」。✅ P = V²/R は純抵抗回路のみ（V が全部 R にかかる前提）。直列では V_R < V。正解は P = I²R = V²R/(R² + X_C²) または P = VI cosφ（cosφ = R/Z）。⚠⑦ ひずみ波で P = VI cosφ をそのまま使う（R06上 問9）

❌ 「公式は1個だから OK」。✅ ひずみ波は各次数（基本波・3次・5次...）で個別計算して合算：P = ΣV_n I_n cosφ_n。実効値も  $I = \sqrt{\sum I_n^2}$ 。力率も「cosφ」という単純角度関係にならない。

## 8拡張：複素電力 S = V·I*（参考・Level 2）🔬 Level 2：複素電力の定義となぜ電流の共役をとるか（R02 問題で出題）

電圧フェーザ  $\dot{V} = V e^{j\theta_V}$ ・電流フェーザ  $\dot{I} = I e^{j\theta_I}$  とするとき、複素電力：

 $\dot{S} = \dot{V}\dot{I}^* = VI e^{j(\theta_V - \theta_I)} = VI(\cos\varphi + j\sin\varphi)$

 $P = \text{Re}[\dot{S}] = VI\cos\varphi, \quad Q = \text{Im}[\dot{S}] = VI\sin\varphi$

- ⑤ 選択肢照合：(E, P) = (20 V, 96 W) = ②📝 まず自分で解答 → 👁 正解・解説を見る過去問は常に即表示にする⚠本問のひっかけポイント（5項目）
- 電力をコンデンサも消費すると思う：X_C は電力を消費しない（P=0）。消費電力は抵抗のみ P=I²R

## 53条件即答チェック（限界的練習ゾーン）📊 このセクションの理解度:◎○△×

P = 100 kW・V = 200 V（線間）・ω = 100π rad/s（50 Hz）固定。cosφ₁ を 0.6 / 0.8 / 0.95 で振り、目標 cosφ₂ = 1.0 への必要 Q_C [kvar] と C [μF] を即答。tan(0°)=0 なので Q_C = P tanφ₁。

即答してから下の解答を開くこと（暗算 30 秒目安）。

| 条件 | tanφ₁ | Q_C [kvar] | C [μF] （V=200, ω=100π） |
| --- | --- | --- | --- |
| ① cosφ₁ = 0.6（遅れ） | sin/cos = 0.8/0.6 ≈ 1.333 | ≈ 133.3 | ≈ 10,610 |
| ② cosφ₁ = 0.8（遅れ） | 0.6/0.8 = 0.75 | 75 | ≈ 5,968 |
| ③ cosφ₁ = 0.95（遅れ） | 0.312/0.95 ≈ 0.329 | ≈ 32.9 | ≈ 2,617 |
解答の導出（3条件まとめて）

① cosφ₁ = 0.6: sinφ₁ = 0.8 ⇒ tanφ₁ = 0.8/0.6 ≈ 1.333・Q_C = 100 × 1.333 ≈ 133.3 kvar・C = Q_C/(ωV²) = 133,300/(100π × 200²) = 133,300/(100π × 40,000) ≈ 10,610 μF

② cosφ₁ = 0.8: sinφ₁ = 0.6 ⇒ tanφ₁ = 0.6/0.8 = 0.75・Q_C = 100 × 0.75 = 75 kvar・C = 75,000/(100π × 40,000) ≈ 5,968 μF

③ cosφ₁ = 0.95: sinφ₁ ≈ 0.312 ⇒ tanφ₁ ≈ 0.329・Q_C ≈ 32.9 kvar・C ≈ 32,900/(100π × 40,000) ≈ 2,617 μF

※ cosφ₁ が小さいほど必要 Q_C と C が劇的に大きくなる（0.6→0.8 で約 1.8倍、0.6→0.95 で約 4倍）。実務でも cosφ ≈ 0.85 まで悪化させると C 増設コストが大きくなることがこれで分かる。⚠典型誤答3択（自己採点用）
- cosφ と sinφ を取り違える（誤）⇒ tanφ = sinφ/cosφ なので逆にすると誤答。sin² + cos² = 1 で必ず sinφ を計算
- kW → W の単位変換忘れ（誤）⇒ Q_C = 100 × 0.75 = 75 を「var」と思って C = 75/(...) で計算すると 1000倍の誤差。P=100kW=100,000W で代入
- 過補償の罠（誤）⇒ 目標を cosφ₂ > 1 と勘違いして Q_C を過大に取る。cosφ ≤ 1 は物理上の上限💡セルフ判定
