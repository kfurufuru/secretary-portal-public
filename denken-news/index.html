ニュース収集コマンド

使用するシェルに合わせてコマンドをコピーして実行してください。Claude が電験関連ニュースを収集し denken-news/news.json を更新します。

CMD（コマンドプロンプト）
cd /d (secretary-portalフォルダ) && claude --dangerously-skip-permissions -p "電験3種に関連する最新ニュースを10件収集し、denken-news/news.json を更新してください。各記事に subjects・priority・exam_angle・quiz を設定すること。"
PowerShell
cd (secretary-portalフォルダ); claude --dangerously-skip-permissions -p "電験3種に関連する最新ニュースを10件収集し、denken-news/news.json を更新してください。各記事に subjects・priority・exam_angle・quiz を設定すること。"

※ claude CLI がインストール済みであること。実行後、ページをリロードすると新記事が反映されます。
